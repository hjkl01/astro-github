# Garage

[Website and documentation](https://garagehq.deuxfleurs.fr/) | [Binary releases](https://garagehq.deuxfleurs.fr/_releases.html) | [Git repository](https://git.deuxfleurs.fr/Deuxfleurs/garage) | [Matrix channel](https://matrix.to/#/%23garage:deuxfleurs.fr)

Garage 是一个 S3 兼容的分布式对象存储服务，专为小到中等规模的自托管地理分布式部署而设计。

Garage 专为由在不同物理位置运行的节点组成的存储集群而设计，以便轻松提供即使某些服务器不可达也能复制数据并保持可用的存储服务。Garage 还专注于轻量级、易于操作，并对机器故障具有高度弹性。

Garage 由 [Deuxfleurs](https://deuxfleurs.fr) 构建，这是一个实验性的小规模自托管服务提供商，自 2020 年首次发布以来一直在生产中使用它。
