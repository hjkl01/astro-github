# Sniffnet

## 项目简介

Sniffnet 是一个跨平台的网络流量监控应用程序，帮助用户舒适地监控互联网流量。它使用 Rust 编写，提供直观的界面和可靠的功能。

## 主要功能

- **网络适配器选择**：选择 PC 的网络适配器进行检查
- **流量过滤**：应用一组过滤器来观察流量
- **统计信息**：查看互联网流量的整体统计数据
- **实时图表**：显示流量强度的实时图表
- **最小化监控**：即使应用程序最小化也能监控网络
- **PCAP 文件支持**：导入和导出综合捕获报告为 PCAP 文件
- **服务识别**：识别 6000+ 上层服务、协议、木马和蠕虫
- **域名和 ASN**：找出与您交换流量的主机的域名和 ASN
- **本地网络**：识别本地网络中的连接
- **地理位置**：发现远程主机的地理位置
- **收藏主机**：保存您喜欢的网络主机
- **实时检查**：实时搜索和检查每个网络连接
- **自定义通知**：设置自定义通知以在定义的网络事件发生时通知您
- **主题支持**：选择最适合您的样式，包括自定义主题支持

## 用法

1. **下载和安装**：从 GitHub Releases 下载适用于您的操作系统的版本（Windows、macOS、Linux）。
2. **安装依赖**：确保安装了所需的系统依赖（参考 Wiki 中的依赖页面）。
3. **启动应用**：运行 Sniffnet，选择网络适配器。
4. **应用过滤器**：根据需要设置过滤器来监控特定流量。
5. **监控流量**：查看实时统计、图表和连接详情。
6. **导出数据**：将捕获的数据导出为 PCAP 文件以供进一步分析。

更多详细用法请参考 [Sniffnet Wiki](https://github.com/GyulyVGC/sniffnet/wiki)。

## 故障排除

- **缺失依赖**：检查并安装所需的系统依赖。
- **渲染问题**：如果界面出现问题，设置环境变量 `ICED_BACKEND=tiny-skia` 以切换到软件渲染器。
