# Syncthing 项目

**GitHub 项目地址:** [https://github.com/syncthing/syncthing](https://github.com/syncthing/syncthing)

## 主要特性

Syncthing 是一个连续文件同步程序。它在两台或更多计算机之间同步文件。我们努力实现以下目标。目标按重要性顺序列出，最重要的首先。这是目标列表的总结版本 - 有关更多评论，请参阅完整的 [目标文档](https://github.com/syncthing/syncthing/blob/main/GOALS.md)。

Syncthing 应该是：

1. **安全免数据丢失**

   保护用户数据至关重要。我们采取一切合理预防措施来避免损坏用户的文件。

2. **安全免受攻击者**

   再次，保护用户数据至关重要。无论我们的其他目标如何，我们绝不能允许用户的数据容易受到未经授权方的窃听或修改。

3. **易于使用**

   Syncthing 应该易于接近、理解和包容。

4. **自动**

   用户交互应该只在绝对必要时才需要。

5. **普遍可用**

   Syncthing 应该在每台普通计算机上运行。我们意识到最新技术并不总是对每个人都可用。

6. **为个人**

   Syncthing 主要是关于授权个人用户安全、可靠和易于使用的文件同步。

7. **其他一切**

   我们关心许多没有列出的东西。只要不与上述陈述目标冲突，优化这些价值观是可以的。

## 主要功能

Syncthing 是一个开源的、去中心化的文件同步工具，旨在提供安全、私密的跨设备文件同步服务。
