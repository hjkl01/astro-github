# Zensical

Zensical is a modern static site generator built by the creators of Material for MkDocs. It allows you to write documentation in Markdown and create a professional static site for your open-source or commercial projects in minutes. The site is searchable, customizable, supports more than 60 languages, and is optimized for all devices.

## Features

- **Markdown-based**: Write your content in simple Markdown format.
- **Searchable**: Built-in search functionality for easy navigation.
- **Customizable**: Highly customizable themes and layouts.
- **Multi-language Support**: Supports over 60 languages.
- **Responsive**: Optimized for all devices, from desktops to mobiles.
- **Fast**: Generates static sites quickly and efficiently.

## Usage

1. **Installation**: Install Zensical via pip or from source.

   ```
   pip install zensical
   ```

2. **Create a Project**: Initialize a new Zensical project.

   ```
   zensical new my-project
   cd my-project
   ```

3. **Write Content**: Add your Markdown files in the `docs` directory.

4. **Build the Site**: Generate the static site.

   ```
   zensical build
   ```

5. **Serve Locally**: Preview the site locally.
   ```
   zensical serve
   ```

For more detailed instructions, visit the [official documentation](https://zensical.org/docs/).

## Compatibility

Zensical is compatible with various platforms and can be integrated with existing workflows. Check the [compatibility page](https://zensical.org/compatibility/) for more information.
