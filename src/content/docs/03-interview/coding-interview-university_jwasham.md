# Coding Interview University

## 功能

Coding Interview University 是一个完整的计算机科学学习计划，旨在帮助用户成为软件工程师。它专注于软件工程面试准备，包括数据结构、算法、系统设计等主题。项目提供了详细的学习路径、书籍推荐、视频资源和实践建议，帮助用户从零开始或补充知识，准备大型科技公司的技术面试，如Amazon、Facebook、Google和Microsoft。

## 用法

1. **开始学习**：按照README中的学习计划顺序，从基础主题开始，如算法复杂度、数据结构等。
2. **使用资源**：观看推荐的视频、阅读书籍、完成编码问题练习。
3. **实践**：在学习每个主题时，通过LeetCode等平台练习编码问题。
4. **跟踪进度**：使用GitHub的issue或fork项目来跟踪学习进度。
5. **准备面试**：学习系统设计、面试技巧，并进行模拟面试。

这个项目适合有一定编程基础但缺乏计算机科学理论知识的用户，或者想系统复习的人。
