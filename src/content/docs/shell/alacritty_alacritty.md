# Alacritty

Alacritty 是一个快速、跨平台的 OpenGL 终端模拟器，具有合理的默认设置，但允许进行广泛的配置。它通过与其他应用程序集成，而不是重新实现其功能，来提供灵活的功能集。目前支持的平台包括 BSD、Linux、macOS 和 Windows。

## 功能

- 快速渲染：使用 GPU 加速的 OpenGL 渲染。
- 跨平台：支持多种操作系统。
- 可配置：通过 TOML 配置文件自定义外观和行为。
- 扩展支持：支持多种插件和主题。

## 用法

1. 安装：从 GitHub Releases 下载二进制文件。
2. 配置：编辑 ~/.config/alacritty/alacritty.toml。
3. 运行：alacritty
