# RyTuneX

RyTuneX 是一个综合性的 Windows 系统优化工具，旨在提升 Windows 系统的性能、隐私和可用性。它使用 WinUI 3 框架和 .NET 8 构建，提供现代且响应式的用户体验。专为 Windows 10 (20H1 及以上版本) 和 Windows 11 设计，无论是游戏玩家还是高级用户，都可以使用它来移除预装应用、管理系统设置和优化性能。

## 功能

- **选择性应用移除**：轻松卸载不需要的预装应用。这允许用户通过移除膨胀软件和不必要的应用程序来清理系统，从而释放宝贵的系统资源并提升整体性能。
- **系统优化**：通过减少资源使用和提升整体系统效率来简化系统。提供从基础到高级的选项，适合初学者和专家用户。
- **隐私增强**：通过阻止遥测和数据收集来提升隐私。RyTuneX 提供工具来禁用 Windows 中固有的跟踪和数据收集功能，确保用户对个人数据有更大的控制权，并更有效地保护隐私。
- **现代设计**：享受由 WinUI 3 驱动的时尚且直观的界面。应用程序具有现代设计，既用户友好又视觉吸引人，使系统优化任务变得简单。界面设计简洁且易于导航，用户可以快速访问所需功能。
- **跨兼容性**：RyTuneX 与 Windows 10 (20H1 及以上版本) 和 Windows 11 兼容，确保在不同 Windows 版本上无缝优化性能。应用程序提供一致的体验，无论操作系统版本如何，用户都可以充分利用其功能。

## 用法

### 安装

#### 选项 1：自动安装（推荐）

1. 打开终端（cmd、PowerShell 或 Windows Terminal）。
2. 运行以下命令：

```
winget install rytunex
```

#### 选项 2：手动安装

1. 下载：获取最新的 [RyTuneX.Setup.zip](https://github.com/rayenghanmi/RyTuneX/releases/latest/download/RyTuneX.Setup.zip)
2. 解压：将下载的 `.zip` 文件解压到您选择的任意位置。
3. 运行安装程序：导航到解压文件夹，双击 `RyTuneX.Setup.exe` 开始安装。
4. 跟随提示：按照屏幕上的指示完成安装。

**重要**：始终从官方 [GitHub releases 页面](https://github.com/rayenghanmi/RyTuneX/releases) 下载 RyTuneX，以确保其真实性和安全性。

### 使用

**警告**：在应用任何优化之前，始终创建还原点。

#### 系统优化

要优化您的系统，请转到 **Optimize** 标签页，并根据您的需求切换可用选项。更改将在系统重启后生效。

#### 去膨胀不需要的应用

在 **Debloat** 标签页，您可以查看已安装应用的列表。选择您想要移除的应用，然后点击 **Uninstall selected App(s)**。可能需要重启来完成过程。

#### 隐私增强

前往 **Privacy** 标签页来配置您的隐私设置。切换开关以启用或禁用影响遥测、广告等的特性。更改将在重启后生效。

#### Windows 功能管理

使用 **Features** 标签页来修改特定的 Windows 功能。切换开关来调整设置，并重启您的系统以应用更改。

#### 系统信息

**Device** 页面提供您系统硬件和操作系统的全面视图。这有助于您了解您的设置，并做出更明智的优化决策。
