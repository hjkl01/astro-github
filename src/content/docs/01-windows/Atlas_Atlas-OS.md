# AtlasOS 项目

**GitHub 项目地址:** [https://github.com/Atlas-OS/Atlas](https://github.com/Atlas-OS/Atlas)

## 主要特性

AtlasOS 是一个开源项目，通过方便地应用隐私、可用性和性能优化来增强 Windows，同时保持功能性和自定义性。主要特性包括：

- **增强隐私**：移除 Windows 中嵌入的大多数遥测，并实施众多组策略以最小化数据收集。
- **优化性能**：在性能和兼容性之间取得平衡，实现众多有意义的更改以改善 Windows 性能和响应性，而不破坏基本功能。
- **安全特性**：允许用户自定义安全，同时告知每个选项的利弊，包括 Windows Defender、SmartScreen、Windows Update 等。
- **增加可用性**：应用许多修改和默认设置，使 Windows 更容易使用，包括移除常用不需要的应用（可重新安装）、配置界面许多方面、禁用广告等。
- **开源和透明**：使用 AME Wizard，使其更容易审计。Playbooks 是可自定义的脚本系统，包含开源二进制文件。
- **法律合规**：不重新分发修改的 Windows ISO，符合 Microsoft Windows 使用条款。

## 功能

- **隐私优化**：移除大多数 Windows 遥测，实施组策略最小化数据收集。
- **性能提升**：改善 Windows 性能和响应性，平衡兼容性。
- **安全自定义**：可选启用/禁用安全功能，如 CPU 缓解、用户账户控制等。
- **可用性改进**：移除不必要的应用，配置界面，禁用广告。
- **透明审计**：使用开源 Playbooks 和工具，便于检查修改。

## 用法

AtlasOS 使用 AME Wizard 应用修改，而不是提供修改的 ISO。请参考官方文档获取详细说明：

- [安装指南](https://docs.atlasos.net/getting-started/installation/)
- [安装 FAQ](https://docs.atlasos.net/install-faq/removed-features/)
- [一般 FAQ](https://docs.atlasos.net/general-faq/atlas-and-security/)

注意：AtlasOS 不是操作系统变体，而是对现有 Windows 的修改。使用前备份数据，适合高级用户。

更多细节请参考仓库的 README 和文档。
