# Lively 项目概述

## 项目地址

[GitHub 项目地址](https://github.com/rocksdanister/lively)

## 主要特性

Lively 是一个开源的动画壁纸系统，将视频、GIF、仿真器、HTML、Web 地址、Shaders、游戏等转换为 Windows 桌面壁纸。当全屏应用/游戏运行时，壁纸会完全暂停播放（0% CPU 和 GPU 使用）。主要特性包括：

- **多种壁纸类型**：支持视频、YouTube 流、Web 页面、Shaders、GIF、游戏等。
- **性能优化**：硬件加速视频播放，全屏时暂停，支持多显示器。
- **易用性**：拖拽文件或网页到窗口设置壁纸。
- **开源免费**：完全开源，无付费功能。
- **社区支持**：Discord 和 Reddit 讨论组。

## 主要功能

- **视频壁纸**：播放 .mp4、mkv、webm 等，支持硬件加速。
- **YouTube 和流**：拖拽 YouTube 链接设置，可调质量。
- **Web 和 HTML**：加载 HTML 或 Web 地址，支持 WebGL、JS、音频响应。
- **Shaders**：运行 GLSL Shaders，支持 Shadertoy URLs。
- **GIF 壁纸**：设置 GIF 作为壁纸。
- **游戏和仿真器**：支持 Unity、Godot 游戏作为壁纸。
- **多显示器**：跨屏、复制或每屏不同壁纸。
- **播放控制**：基于应用规则暂停，音频静音等。

## 用法指南

1. **下载**：
   - 从 Releases 下载安装器或便携版（v0.9.6.0）。
   - 安装器：lively_setup_x86_full_v0960.exe
   - 便携版：lively_portable_x86_full_v0960.zip（需 VC++ Redist）

2. **安装**：
   - 运行安装器，可能有 Smartscreen 警告（安全）。

3. **使用**：
   - 打开 Lively，拖拽文件或网页到窗口设置壁纸。
   - 浏览库预览壁纸。
   - 设置：调整质量、规则等。

4. **高级用法**：
   - 创建 Lively-zip 分享壁纸。
   - 自定义 Web 壁纸开发。
   - 故障排除：检查日志。

Lively 适合喜欢动态壁纸的用户。更多细节请参考项目 Wiki。
