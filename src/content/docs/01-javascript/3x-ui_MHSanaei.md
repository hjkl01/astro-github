# 3x‑UI by MHSanaei

**项目地址**: <https://github.com/MHSanaei/3x-ui>

## 项目简介

3X-UI 是一个高级、开源的基于 Web 的控制面板，专为管理 Xray-core 服务器而设计。它提供了用户友好的界面，用于配置和监控各种 VPN 和代理协议。作为 X-UI 项目的增强分支，3X-UI 提供了改进的稳定性、更广泛的协议支持和附加功能。

## 主要特性

- **用户友好的界面**：用于配置和监控 Xray-core 服务器。
- **协议支持**：支持各种 VPN 和代理协议。
- **稳定性改进**：作为 X-UI 的增强分支，提供更好的稳定性。
- **附加功能**：包括路由规则、用户管理等。
- **开源**：基于 GPL-3.0 许可证。

## 核心功能

| 功能     | 描述                          |
| -------- | ----------------------------- |
| 面板管理 | Web 界面管理 Xray 配置        |
| 用户管理 | 添加、编辑、删除用户          |
| 流量统计 | 监控用户和服务器流量          |
| 路由规则 | 配置自定义路由规则            |
| 支持协议 | V2Ray、Trojan、Shadowsocks 等 |

## 用法示例

1. **快速安装**

   ```bash
   bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
   ```

2. **访问面板**
   - 默认端口：2053
   - 用户名/密码：admin/admin

3. **配置 Xray**
   - 在面板中添加入站/出站规则
   - 配置用户和协议

> **重要提醒**：此项目仅供个人使用，请勿用于非法目的。
