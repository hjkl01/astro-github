## 常见命令

| 选项 | 含义 |
|------|------|
| `--write` | 直接修改文件 |
| `--check` | 检查文件是否已格式化 |
| `--list-different` | 列出不符合格式的文件 |
| `--config` | 指定配置文件路径 |
| `--ignore-path` | 指定 `.prettierignore` 路径 |
