## 功能

- **学习路线规划**：30 天学习计划，适合零基础或已有基础的开发者。
- **示例代码/练习**：每一节都包含实现代码及练习，让学习更有成效。
- **项目扩展**：学习者可通过改写示例或自行扩展来构建更大阐述的项目。
- **版本控制与社区协作**：使用 GitHub 进行版本管理，方便贡献代码、提交 Issue 与 Pull Request。

## 用法

1. **克隆仓库**

   ```bash
   git clone https://github.com/Asabeneh/30-Days-Of-JavaScript.git
   cd 30-Days-Of-JavaScript
   ```

2. **按日阅读**
   - 浏览 `day1`、`day2` … `day30` 文件夹。
   - 运行示例代码：
     ```bash
     node dayX/example.js
     ```
     或在浏览器中打开对应的 `.html` 文件。

3. **完成练习**
   - 根据 README 或文件说明完成练习题。
   - 可自行改写代码并尝试新的实现。

4. **提交 PR**
   - 若发现错误或想改进示例，直接 fork → commit → PR。
