# LunarBar

> **项目地址**：<https://github.com/LunarBar-app/LunarBar>

## 简介  
LunarBar 是一款 macOS 菜单栏应用，专注于提供月相、农历及相关天文信息。它以简洁直观的图标展示月相变化，点击后可查看详细的月相数据和农历信息，适合需要随时关注月相变化的用户。

## 主要特性  

| 特性 | 说明 |
|------|------|
| **月相图标** | 菜单栏中显示当前月相的图标，随月相实时变化。 |
| **详细月相信息** | 点击图标弹出窗口，展示：<br>• 月龄（天）<br>• 亮度（%）<br>• 当前相位（如新月、上弦月等）<br>• 下一相位与到达时间 |
| **农历信息** | 显示当前农历日期、生肖、节气等。 |
| **天文时间** | 提供月升月落时间、日出日落时间等。 |
| **自定义设置** | 可在偏好设置中调整：<br>• 位置（左/右/居中）<br>• 主题（深色/浅色）<br>• 频率更新间隔<br>• 语言（中英双语） |
| **通知提醒** | 支持设置月相变化提醒，提醒用户关注特定相位。 |
| **跨平台** | 仅支持 macOS，兼容 macOS 10.15 (Catalina) 及以上版本。 |

## 功能概览  

1. **月相实时更新**  
   - 通过内置天文算法实时计算月相，刷新频率可在设置中自定义。

2. **详细视窗**  
   - 点击菜单栏图标弹出窗口，提供月相、农历、天文时刻等信息。  
   - 窗口支持拖拽、缩放与自定义布局。

3. **通知与提醒**  
   - 可开启系统通知，在特定月相（如满月、下弦月）时提醒。  
   - 提供自定义提醒时间与重复规则。

4. **偏好设置**  
   - `Appearance`：选择深色或浅色主题。  
   - `Position`：菜单栏图标位置。  
   - `Language`：中英文切换。  
   - `Update Frequency`：设置信息更新间隔。

5. **快捷键**  
   - 默认快捷键 `⌘+Option+L` 打开/关闭详细视窗。  
   - 可在偏好设置中自定义。

## 用法  

1. **下载安装**  
   - 从 GitHub Release 页面下载最新 `.dmg` 包，双击安装。  
   - 或使用 Homebrew Cask：  
     ```bash
     brew install --cask lunarbar
     ```

2. **启动**  
   - 安装后，LunarBar 会自动在菜单栏显示月相图标。  
   - 如未自动启动，可在 `Launchpad` 或 `应用程序` 中打开。

3. **查看信息**  
   - 点击菜单栏图标，弹出窗口显示详细信息。  
   - 右键或长按图标可进入偏好设置。

4. **自定义**  
   - 打开 `LunarBar → Preferences`（或 `⌘+ ,`）进入设置。  
   - 根据需要调整主题、位置、语言、更新频率等。

5. **更新**  
   - 通过 GitHub Release 或 Homebrew 自动更新。  
   - 更新后会提示重新启动以应用新版本。
