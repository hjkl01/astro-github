# JS破解逆向爬虫项目

**项目地址：** [https://github.com/LoseNine/Crack-JS-Spider](https://github.com/LoseNine/Crack-JS-Spider)

## 主要特性

- **JS反爬虫加密参数破解**：专注于破解Python爬虫过程中遇到的JS加密问题。
- **多种网站支持**：已破解多个知名网站的JS加密参数，包括登录、API调用等。
- **逆向工程教程**：提供Chrome调试技巧、JS解密与代码AST混淆还原的教程。
- **开源免费**：MIT许可，允许自由使用、修改和分发。
- **学习资源**：配套书籍《反爬虫JS安全防护与逆向分析手册》、《Android应用安全实战》。

## 主要功能

- **破解案例**：
  - 2022年：极验滑块w、QQ音乐sign、拼多多anti_content、boss直聘zp_token、知乎x-zse-96、酷狗kg_mid/dfid、唯品会mars_cid
  - 2020年：中国裁判文书网、B站登录、房天下登录、淘宝登录、天安保险登录、QQ音乐sign、一键解析、财联社登录
  - 2019年：微博登录、有道翻译、网易登录、微信公众号登录、空中网登录、今目标登录、云就业登录、WPS登录、全国中等职业学校登录、重庆科技资源共享登录、网易云音乐下载
- **JS代码获取**：提供获取关键参数的JS代码。
- **反混淆工具**：用于学习AST原理与还原混淆实战。

## 用法

1. **克隆仓库**：

   ```
   git clone https://github.com/LoseNine/Crack-JS-Spider.git
   cd Crack-JS-Spider
   ```

2. **选择案例**：根据需要破解的网站，选择对应的目录（如QQ音乐、淘宝等）。

3. **运行代码**：按照目录内的说明运行相应的JS或Python代码，获取破解的参数。

4. **学习教程**：参考配套教程《反爬虫JS安全防护与逆向分析手册》学习Chrome调试和JS解密技巧。

5. **自定义破解**：基于现有案例，修改代码以适应新的JS加密场景。

注意：仅用于合法的研究和学习目的，请遵守相关法律法规。
