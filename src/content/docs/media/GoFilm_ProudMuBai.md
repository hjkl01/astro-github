# GoFilm

一个基于 vue 和 gin 实现的在线观影网站

效果展示: [点击访问演示站点](https://m.mubai.link/)

新版本测试访问站点: [测试内容站点](http://113.44.5.201/index)

## 项目地址

[https://github.com/ProudMuBai/GoFilm](https://github.com/ProudMuBai/GoFilm)

## 简介

**GoFilm**

项目采用 vite + vue 作为前端技术栈，使用 ElementPlus 作为 UI 框架进行开发

后端程序使用 Gin + gorm + go-redis 等相关框架提供接口服务，使用 gocolly 和 robfig/cron 进行公共影视资源采集和定时更新功能

## 主要特性

- **多播放源自动采集**：自动从多个在线影视网站采集资源，支持定时更新。
- **现代化前端**：基于 Vue 3 和 Element Plus 的响应式界面，支持移动端和 PC 端。
- **高效后端**：使用 Gin 框架提供 RESTful API，支持数据库操作和缓存。
- **管理后台**：提供后台管理功能，包括采集站点管理、定时任务、网站信息配置等。
- **开源免费**：MIT 许可证，完全开源，欢迎贡献。

## 主要功能

- **在线观影**：支持电影、电视剧的在线播放，提供多种播放源。
- **影片搜索**：支持关键词搜索、分类浏览。
- **用户界面**：首页轮播、影片详情、播放页面等。
- **后台管理**：登录后台管理采集、更新、分类等。
- **移动端支持**：响应式设计，支持移动设备观看历史等。
