# 什么是 Live Agent Studio？  
[Live Agent Studio](https://studio.ottomator.ai) 是由 [oTTomator](https://ottomator.ai) 开发的社区驱动平台，供用户探索前沿 AI 代理并学习如何将其应用于个人或企业！平台上的所有代理均为开源，未来将涵盖大量使用场景。  

该平台的目的是打造一个教育平台，帮助用户学习如何利用 AI 实现创新功能，同时提供实用价值，让用户愿意直接使用这些代理。  

目前平台处于测试阶段，可能会出现负载高时响应延迟、代理库快速扩展、以及 Cole Medin 的 YouTube 频道将陆续增加内容。  

# 这个仓库的作用？  
该仓库包含 Live Agent Studio 上所有代理的源代码和工作流程 JSON！所有新增代理都将在此开源，以便社区共同构建和学习。  

## 代币（Tokens）  
平台上的大多数代理使用时需消耗代币，可通过平台购买。新用户注册后会获得初始代币，可免费试用代理。代币费用用于支付平台托管代理所需的 LLM 成本。  

[购买代币](https://studio.ottomator.ai/pricing)  

## 未来计划  
随着平台发展，Live Agent Studio 将成为 AI 代理领域的权威信息源。任何新 AI 技术、突破性代理研究或工具库都会通过平台代理展示。  

## 常见问题（FAQ）  
### 如何将自己开发的代理展示在 Live Agent Studio？  
参考 [开发者指南](https://studio.ottomator.ai/guide)，并查看 [示例 n8n 代理](~sample-n8n-agent~) 和 [示例 Python 代理](~sample-python-agent~) 作为起点。  

### 使用代理需要消耗多少代币？  
每代理按提示收费，具体取决于代理的复杂度（如使用的 LLM 规模、调用次数或付费 API）。  

### 如何交流讨论代理并获取帮助？  
加入 [Think Tank 社区](https://thinktank.ottomator.ai) 发帖提问。
