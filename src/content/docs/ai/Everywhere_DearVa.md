## 项目简介

Everywhere 是一个上下文感知的桌面 AI 助手，能够智能响应并无缝集成多个大型语言模型 (LLM) 和 MCP 工具。与传统 AI 工具不同，Everywhere 可以即时感知和理解屏幕上的任何内容，无需截图、复制或切换应用。只需按下快捷键，即可在任何地方获得所需的 AI 帮助。

## 主要功能

### 🌟 体验示例

- **故障排除专家**：遇到错误消息时，直接在错误消息上调用 Everywhere，询问“这是什么错误？如何解决？”即可获得解决方案。
- **网页快速摘要**：浏览长篇文章时，在网页上调用 Everywhere，命令“给我一个 3 点摘要”，即可快速获取要点。
- **即时翻译**：遇到外语文本时，在文本上调用 Everywhere，命令“翻译成英语”，即可获得即时翻译。
- **邮件草稿协助**：撰写商务邮件时，在草稿上调用 Everywhere，命令“让这个听起来更专业”，即可将随意文本转换为专业商务沟通。

### 🛠️ 技术特性

- **LLM 支持**：支持 OpenAI、Anthropic (Claude)、Google (Gemini)、DeepSeek、Moonshot (Kimi)、OpenRouter、SiliconCloud、Ollama 等多种模型，以及自定义端点和官方 API。
- **工具集成**：集成 Web 浏览器、文件系统、终端、Everything (Windows) 等工具，以及 MCP 工具。
- **交互体验**：现代毛玻璃 UI、上下文感知调用、键盘快捷键、Markdown 渲染等。
- **平台支持**：目前支持 Windows，macOS 和 Linux 即将支持。
- **语言支持**：支持简体中文、English、Deutsch、Español、Français、Italiano、日语、韩语、Русский、Türkçe、中文 (繁體)、中文 (繁體 香港特別行政區) 等多种语言。

## 系统要求

- **Windows**：最低版本 10.0.19041.0
- **macOS**：即将支持
- **Linux**：即将支持

## 安装和使用

### 安装

- **Windows**：从 [GitHub Releases](https://github.com/DearVa/Everywhere/releases/latest) 下载最新版本。`Everywhere-Windows-x64-Setup-vx.x.x.exe` 为安装包，`Everywhere-Windows-x64-vx.x.x.zip` 为免安装包，内容相同。
- **macOS 和 Linux**：目前不支持，但即将推出。
