# Vibe Kanban

> 项目地址: https://github.com/BloopAI/vibe-kanban

## 简介

Vibe Kanban 是一款轻量级、响应式的看板管理工具，提供拖拽式任务管理、列与卡片自定义、标签过滤、搜索、导入/导出等核心功能，适用于个人或团队的项目进度可视化。

## 主要特性

- **拖拽式任务管理**  
  在列与列之间、列内任意位置拖拽任务卡片，实时更新任务状态。

- **多列与自定义列**  
  默认三列（待办、进行中、已完成），可随时新增、删除或重命名列。

- **任务卡片详情**  
  每张卡片可展示标题、描述、优先级、标签、负责人、创建/更新时间等信息。

- **标签与颜色**  
  支持自定义标签，并为标签分配颜色，便于快速识别。

- **搜索与过滤**  
  提供关键词搜索以及按标签、负责人、优先级等字段过滤任务。

- **移动端适配**  
  响应式布局，兼容桌面与移动设备。

- **数据持久化**  
  默认使用 `localStorage` 存储看板状态，亦可通过后端 API 进行持久化。

- **导入/导出**  
  支持将看板数据导出为 JSON 文件，或从 JSON 文件导入恢复看板。

## 功能说明

| 功能 | 操作方式 | 说明 |
|------|----------|------|
| 创建任务 | `+ 新建任务` 按钮 | 弹出表单填写标题、描述等，保存后即生成卡片 |
| 编辑任务 | 点击卡片进入编辑模式 | 修改任何字段后保存即可 |
| 删除任务 | 卡片右上角删除图标或列表管理界面 | 一键删除 |
| 拖拽移动 | 拖拽卡片到目标列或列内 | 实时更新列状态 |
| 列管理 | 列标题旁编辑图标 | 修改列名、删除列、添加新列 |
| 标签管理 | 卡片编辑界面添加/删除标签 | 也可在设置中自定义标签颜色 |
| 搜索与过滤 | 搜索框 + 过滤器面板 | 根据关键词、标签、负责人、优先级筛选 |
| 导入/导出 | 设置页面导入/导出按钮 | 导出为 JSON，导入从 JSON 恢复 |

## 用法

1. **克隆仓库**

   ```bash
   git clone https://github.com/BloopAI/vibe-kanban.git
   cd vibe-kanban
   ```

2. **安装依赖**

   ```bash
   npm install   # 或 yarn install
   ```

3. **启动开发服务器**

   ```bash
   npm run dev   # 或 yarn dev
   ```

   浏览器访问 `http://localhost:5173`（或终端提示的端口）即可使用。

4. **构建发布包**

   ```bash
   npm run build   # 或 yarn build
   ```

   生成的静态文件位于 `dist/`，可直接部署至任意静态服务器。

5. **配置后端（可选）**

   若需将数据持久化到服务器，修改 `src/api` 中的接口配置，或在 `src/store` 中替换本地存储实现。

## 目录结构
