# Graphite

**项目地址**  
https://github.com/GraphiteEditor/Graphite

Graphite 是一款开源的矢量图形编辑器，旨在为设计师和艺术家提供一套完整、直观、高效的绘图工具。该项目基于 Electron、React 以及 Fabric.js 开发，兼容 Windows、macOS 与 Linux。

## 主要特性

| 功能 | 说明 |
|------|------|
| **矢量绘图** | 支持自由绘制、形状工具（矩形、圆形、直线、曲线）、路径编辑与贝塞尔曲线 |
| **图层管理** | 图层面板支持添加/删除/重命名、锁定、隐藏、顺序调整、组/解组、图层样式 |
| **文本编辑** | 富文本编辑、字体、大小、颜色、对齐方式、文字路径 |
| **颜色与材料** | 颜色选择器、渐变、透明度、填充与描边样式 |
| **导入导出** | SVG、PNG、JPG、BMP、TIFF、PDF 等格式 |
| **插件扩展** | 通过插件 API 可扩展功能与界面 |
| **历史记录** | 撤消/重做（无限层级） |
| **快捷键** | 常用绘图与编辑操作快捷键映射 |
| **协作模式**（预览） | 与云端同步或多人协作（正在开发中） |

## 功能使用示例

```bash
# 克隆仓库
git clone https://github.com/GraphiteEditor/Graphite.git

# 进入项目目录
cd Graphite

# 安装依赖
npm install

# 打开应用
npm run dev
```

打开后即可：

1. **绘制图形** – 选择工具栏中的形状或自由绘图工具，单击并拖拽即可绘制。  
2. **编辑图层** – 在右侧图层面板中管理图层，支持多选、复制、粘贴、删除、调整顺序。  
3. **设计文本** – 选择文字工具，单击画布添加文字，使用属性栏调整字体、大小、颜色。  
4. **导出文件** – 菜单栏 → 文件 → 导出 → 可选 SVG 或 PNG 等格式；保存为本地文件。  

## 开发与贡献

- **代码结构**：前端采用 React + TypeScript；核心绘图逻辑封装在 `src/core`、SDK 在 `src/api`。  
- **运行交付**：`npm run build` 打包后即可在 `dist/` 生成可执行文件。  
- **贡献说明**：请先阅读 `CONTRIBUTING.md`，提交 PR 时需遵循代码规范与测试要求。
