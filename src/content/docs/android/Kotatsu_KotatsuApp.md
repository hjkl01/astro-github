# Kotatsu

Kotatsu 是一个免费的开源安卓漫画阅读器，具有内置的在线内容来源。它支持从多个漫画网站浏览、搜索和阅读漫画，并提供离线下载功能。

## 主要功能

- **在线漫画目录**：支持 1200+ 漫画来源，包括各种漫画网站。
- **搜索和过滤**：按名称、类型和其他过滤器搜索漫画。
- **收藏管理**：收藏夹按用户定义类别组织，支持阅读历史、书签和隐身模式。
- **离线阅读**：下载漫画并离线阅读，支持第三方 CBZ 档案。
- **用户界面**：干净方便的 Material You UI，针对手机、平板和桌面优化。
- **阅读器**：标准和 Webtoon 优化的可定制阅读器，支持手势操作。
- **通知和推荐**：新章节通知、更新 feed 和漫画推荐（带过滤器）。
- **集成服务**：与 Shikimori、AniList、MyAnimeList 和 Kitsu 等漫画跟踪服务集成。
- **安全**：密码或指纹保护访问应用。
- **同步**：自动同步应用数据与其他设备在同一账户。
- **兼容性**：支持 Android 6.0+ 的设备。

## 用法

1. **安装**：从 GitHub Releases 下载 APK 文件，或从 F-Droid 等应用商店安装。
2. **启动应用**：打开 Kotatsu 应用。
3. **浏览漫画**：选择漫画来源，浏览或搜索漫画。
4. **阅读**：点击漫画开始阅读，支持在线或离线模式。
5. **设置**：在设置中自定义阅读器、同步账户等。

注意：该项目已于 2025 年 11 月 4 日存档，不再提供支持。
