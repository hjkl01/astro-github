# WebViewTvLive

## 简介

WebViewTvLive 是一个使用腾讯X5 WebView开发的Android电视直播应用。它通过一键打开官方网页，提供方便快捷的观看体验。应用嵌套展示电视台公开的直播页面，所有内容来自原网站，版权归原网站所有。

## 功能

- **自动更新频道列表**：自动同步最新的频道信息。
- **多直播源**：支持多种直播源选择。
- **应用自定义设置**：允许用户进行个性化配置。
- **多设备适配**：兼容手机、平板、电视和车机。
- **数字键换台**：使用数字键快速切换频道。
- **上下键换台反转**：支持上下键反转换台操作。
- **TBS内核选择**：用户可自主选择是否启用TBS内核。

## 用法

1. **下载安装**：从GitHub Releases页面下载最新APK文件。
2. **安装应用**：在Android设备上安装APK（可能需要允许未知来源安装）。
3. **打开应用**：启动WebViewTvLive应用。
4. **选择频道**：浏览频道列表，选择想要观看的直播频道。
5. **观看直播**：应用将加载官方直播页面，开始播放。
6. **换台操作**：使用数字键或上下键切换频道。

注意：首次加载频道可能需要等待一定时间。应用不支持选择视频清晰度，大部分为24FPS。

## 设备要求

### 最低配置

- CPU：四核 ARM Cortex-A53（≥1.2GHz）
- 内存：2GB
- 系统版本：Android 5.0
- WebView版本：WebView 78

### 推荐配置

- CPU：四核或以上 ARM Cortex-A55 / A76 混合架构
- 内存：4GB 及以上
- 系统版本：Android 11 及以上
- WebView版本：WebView 120 及以上
