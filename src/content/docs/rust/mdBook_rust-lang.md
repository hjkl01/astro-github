mdBook 是一个用于从 Markdown 文件创建现代在线书籍的工具，类似于 Gitbook，但使用 Rust 实现。

## 功能

- 从 Markdown 文件生成书籍
- 支持现代在线书籍格式
- 提供用户指南作为演示示例

## 用法

1. 安装 mdBook：通过 Cargo 安装 `cargo install mdbook`
2. 创建新书籍：`mdbook init my-book`
3. 构建书籍：`mdbook build`
4. 预览书籍：`mdbook serve`

更多详细信息请参考 [用户指南](https://rust-lang.github.io/mdBook/)。
