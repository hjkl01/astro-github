# Server (LandSandBoat)

## 功能

LandSandBoat 是一个开源的 Final Fantasy XI (FFXI) 服务器模拟器。它允许玩家在私人服务器上体验 FFXI 游戏，包括战斗、任务、探索等核心功能。该项目使用 C++ 和 Lua 编写，支持自定义脚本和模块，以扩展游戏内容。

主要功能包括：

- 完整的 FFXI 游戏世界模拟
- 支持多人在线游戏
- 自定义任务和事件脚本
- 数据库驱动的游戏数据管理
- 社区驱动的开发和更新

## 用法

### 快速开始

1. 克隆仓库：`git clone https://github.com/LandSandBoat/server.git`
2. 按照 [快速开始指南](https://github.com/LandSandBoat/server/wiki/Quick-Start-Guide) 设置环境。
3. 编译项目：使用 CMake 构建服务器。
4. 配置数据库：导入 SQL 文件到 MariaDB 或 MySQL。
5. 运行服务器：启动游戏服务器和相关服务。
