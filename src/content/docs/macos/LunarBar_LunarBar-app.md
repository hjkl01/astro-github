# LunarBar

> **项目地址**：<https://github.com/LunarBar-app/LunarBar>

## 简介

LunarBar 是一款完全免费且开源的 macOS 状态栏极简日历应用，支持农历、公共假日、系统日历及提醒等功能。它以简洁的设计展示日期信息，点击后可查看详细的农历和日历数据，适合需要随时查看日期的用户。

## 主要特性

| 特性             | 说明                                                                                                    |
| ---------------- | ------------------------------------------------------------------------------------------------------- |
| **农历显示**     | 菜单栏中显示当前农历日期，包括生肖、节气等。                                                            |
| **公共假日**     | 显示法定假日和节假日信息。                                                                              |
| **系统日历集成** | 与 macOS 系统日历同步，显示事件提醒。                                                                   |
| **提醒功能**     | 支持设置日期相关提醒。                                                                                  |
| **自定义设置**   | 可在偏好设置中调整：<br>• 位置（左/右/居中）<br>• 主题（深色/浅色）<br>• 更新频率<br>• 语言（中英双语） |
| **跨平台**       | 仅支持 macOS，兼容 macOS 15.0 及以上版本。                                                              |

## 功能概览

1. **农历实时更新**
   - 实时显示农历日期、生肖和节气信息。

2. **详细视窗**
   - 点击菜单栏图标弹出窗口，提供农历、日历、假日等信息。
   - 窗口支持拖拽和自定义布局。

3. **通知与提醒**
   - 可开启系统通知，在重要日期时提醒。
   - 支持自定义提醒。

4. **偏好设置**
   - `Appearance`：选择深色或浅色主题。
   - `Position`：菜单栏图标位置。
   - `Language`：中英文切换。
   - `Update Frequency`：设置信息更新间隔。

5. **快捷键**
   - 默认快捷键可在设置中自定义。

## 用法

1. **下载安装**
   - 从 GitHub Releases 页面下载最新 `LunarBar.dmg` 文件，双击打开，将 `LunarBar.app` 拖拽至 `Applications` 文件夹。
   - 或使用 Homebrew Cask：
     ```bash
     brew install --cask lunarbar
     ```

2. **启动**
   - 安装后，LunarBar 会自动在菜单栏显示日历图标。
   - 如未自动启动，可在 `Launchpad` 或 `应用程序` 中打开。

3. **查看信息**
   - 点击菜单栏图标，弹出窗口显示农历、假日等详细信息。
   - 右键或长按图标可进入偏好设置。

4. **自定义**
   - 打开 `LunarBar → Preferences`（或 `⌘+ ,`）进入设置。
   - 根据需要调整主题、位置、语言、更新频率等。

5. **更新**
   - LunarBar 会自动检查更新，你也可以在应用内手动检查更新，或在 GitHub Releases 页面浏览历史版本。
