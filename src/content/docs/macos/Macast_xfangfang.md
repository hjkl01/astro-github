# Macast 项目

## 项目地址

[https://github.com/xfangfang/Macast](https://github.com/xfangfang/Macast)

## 主要特性

Macast 是一个跨平台的开源应用，使用 mpv 作为 DLNA Media Renderer，主要用于从移动设备推送媒体内容到电脑上播放。它提供简单易用的界面，帮助用户实现跨设备媒体共享。核心特性包括：

- **DLNA 支持**：作为 DLNA 媒体渲染器，接收来自手机或其他 DLNA 客户端的视频、图片和音乐推送。
- **跨平台**：支持 macOS、Windows 和 Linux 系统。
- **mpv 播放器**：使用强大的 mpv 播放器作为后端，支持多种媒体格式。
- **插件支持**：支持第三方播放器插件，如 IINA 和 PotPlayer。
- **菜单栏应用**：在系统托盘/任务栏显示图标，便于操作。
- **开源免费**：基于 GPL-3.0 许可，完全开源。
- **轻量级**：资源占用低，适合日常使用。

## 主要功能

- **接收媒体推送**：从 DLNA 客户端（如手机应用）推送视频、音乐、图片到电脑播放。
- **媒体播放控制**：支持播放、暂停、快进等控制。
- **第三方播放器**：通过插件支持 IINA、PotPlayer 等播放器。
- **自定义配置**：修改快捷键或 mpv 配置。
- **日志与调试**：提供日志帮助排查问题。

## 用法

1. **下载与安装**：
   - 从 GitHub Releases 页面下载适用于 macOS 的安装包。
   - 或使用 pip：`pip install macast`，然后运行 `macast-gui`。

2. **启动应用**：
   - 打开应用后，菜单栏/任务栏会显示图标，表示 DLNA 渲染器已启动。

3. **推送媒体**：
   - 使用手机上的 DLNA 客户端应用，选择媒体文件，推送至 Macast。
   - Macast 会使用 mpv 播放接收到的媒体。

4. **高级用法**：
   - 加载插件支持第三方播放器。
   - 修改 mpv 配置或快捷键。
   - 查看日志排查问题。

Macast 支持 macOS 10.14 及以上版本。
