## Usage

### Running the Scraper

```bash
# Scrape trending repositories and update urls.txt
python github_trending_scraper.py

# Generate markdown files for new projects
python main.py

# Categorize existing files
python main.py cate
```

### Building the Site

```bash
pnpm install
pnpm dev      # Start development server
pnpm build    # Build for production
pnpm preview  # Preview build
```
