Deep Chat 是一个完全可定制的 AI 聊天组件，只需一行代码即可注入您的网站。无论您想创建一个利用 ChatGPT 等流行 API 的聊天机器人，还是连接到您自己的自定义服务，这个组件都能做到！探索 deepchat.dev 查看所有可用功能、如何使用它们、示例等！

## 主要特性

- **连接任何 API**：支持连接到任何 API。
- **头像和名称**：支持头像和名称。
- **发送/接收文件**：支持发送和接收文件。
- **摄像头拍照**：通过摄像头捕获照片。
- **麦克风录音**：通过麦克风录制音频。
- **语音到语音通信**：支持语音到语音通信。
- **语音到文本输入**：支持语音到文本的消息输入。
- **文本到语音响应**：支持听取消息响应的文本到语音。
- **MarkDown 和自定义元素支持**：帮助结构化文本和渲染代码。
- **介绍面板和动态模态**：帮助描述用户功能。
- **连接到 20+ 流行 AI API**：如 OpenAI 或 Claude 直接从浏览器。
- **与语音到语音模型通信**：支持语音到语音模型。
- **支持所有主要 UI 框架/库**：支持所有主要 UI 框架/库。
- **在浏览器上托管模型**：在浏览器上托管模型。
- **焦点模式**：仅显示最新的消息。
- **一切都可定制**：一切都可定制！

## 用法

### 开始使用

```bash
npm install deep-chat
```

如果使用 React，安装以下内容：

```bash
npm install deep-chat-react
```

只需将以下内容添加到您的标记中：

```html
<deep-chat></deep-chat>
```

确切的语法将根据您选择的框架而有所不同（[请参阅此处](https://deepchat.dev/examples/frameworks)）。
