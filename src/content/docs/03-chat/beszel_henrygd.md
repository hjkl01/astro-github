# Beszel 项目

## 项目地址

[GitHub 项目地址](https://github.com/henrygd/beszel)

## 主要特性

Beszel 是一个轻量级的服务器监控中心，旨在提供简单而强大的服务器监控功能。其核心特性包括：

- **轻量级设计**：资源占用低，易于部署和运行。
- **多平台支持**：支持多种操作系统和架构。
- **实时监控**：提供实时的服务器状态监控，包括 CPU、内存、磁盘等指标。
- **易于使用**：界面简洁，用户友好。
- **开源免费**：完全开源，MIT 许可。

## 主要功能

- **系统监控**：监控服务器的 CPU 使用率、内存使用率、磁盘空间等。
- **网络监控**：监控网络流量和连接状态。
- **告警通知**：支持自定义告警规则和通知方式。
- **历史数据**：存储历史监控数据，便于分析。
- **多服务器支持**：可以监控多个服务器。

## 用法

1. **安装**：
   - 从 GitHub Releases 下载对应平台的二进制文件。
   - 或者使用 Docker：`docker run -d -p 8090:8090 -v /path/to/data:/beszel_data henrygd/beszel`

2. **配置**：
   - 启动后，访问 Web 界面进行配置。
   - 添加要监控的服务器。

3. **监控**：
   - 查看实时监控数据。
   - 设置告警规则。

更多详情请参考项目 README 文件。
