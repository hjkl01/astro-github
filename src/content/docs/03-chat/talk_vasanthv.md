# Ahey 项目

## 项目地址

[GitHub 项目地址](https://github.com/vasanthv/ahey)

## 主要特性

Ahey 是一个免费的、点对点的群组视频通话应用，专为 Web 设计，无需注册、无需下载。其核心特性包括：

- **点对点视频通话**：使用 WebRTC 实现低延迟的直接视频流。
- **群组通话**：支持多人视频通话，使用网格拓扑。
- **无注册**：无需账户或密码，即可加入或创建通话。
- **跨平台**：在所有主要浏览器中工作。
- **免费**：100% 免费，基于 Web。

## 主要功能

- **视频通话**：高清视频通话，支持多人。
- **音频通话**：内置音频支持。
- **文本聊天**：在通话中发送文本消息。
- **屏幕共享**：允许共享屏幕。
- **易嵌入**：可嵌入到任何网站或应用中。

## 用法

1. **访问应用**：
   - 打开浏览器，访问 ahey.io 或自托管实例。

2. **创建或加入通话**：
   - 输入频道 ID 创建或加入通话。
   - 分享 URL 给其他人。

3. **开始通话**：
   - 允许摄像头和麦克风访问。
   - 开始视频通话。

更多详情请参考项目 README 文件。
