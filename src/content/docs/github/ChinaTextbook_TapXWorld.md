# ChinaTextbook

ChinaTextbook 是一个开源项目，由 TapXWorld 创建，旨在提供中国小学、初中、高中和大学的 PDF 教材资源。该项目收集了义务教育阶段的数学教材，帮助促进教育平等和消除地区间的教育贫困，同时也为海外华人提供国内教育资源。

## 功能

- **免费教材下载**：提供小学、初中、高中和大学数学教材的 PDF 文件。
- **开源共享**：所有资源免费开源，鼓励教育资源的普及。
- **文件合并工具**：对于超过 50MB 的文件，提供自动合并工具来处理拆分的文件。
- **社区支持**：通过 Telegram 社区获取最新动态和分享想法。

## 用法

1. **下载教材**：
   - 访问 GitHub 仓库：https://github.com/TapXWorld/ChinaTextbook
   - 浏览文件夹（如小学、初中、高中、大学），选择所需的 PDF 文件下载。

2. **合并拆分文件**（如果适用）：
   - 下载合并程序 `mergePDFs-windows-amd64.exe` 从 https://github.com/TapXWorld/ChinaTextbook-tools/releases
   - 将程序和拆分文件放在同一目录下，双击运行程序即可自动合并。

3. **重新下载**：
   - 如果网络条件允许，使用 [tchMaterial-parser](https://github.com/happycola233/tchMaterial-parser) 项目重新下载资源。
