## NocoBase 简介

NocoBase 是一个最可扩展的 AI 驱动的无代码/低代码平台，用于构建商业应用程序和企业解决方案。它提供了完全控制、无限扩展性和 AI 协作能力，帮助团队快速适应并大幅降低成本。

## 主要功能

### 1. 数据模型驱动，而非表单/表格驱动

NocoBase 采用数据模型驱动的方法，将数据结构与用户界面完全分离，解锁无限可能性。

- UI 和数据结构完全解耦
- 可以为同一表或记录创建任意数量和形式的多个块和操作
- 支持主数据库、外部数据库和第三方 API 作为数据源

### 2. AI 员工，集成到您的业务系统中

与独立的 AI 演示不同，NocoBase 允许您将 AI 能力无缝嵌入到界面、工作流和数据上下文中，使 AI 在实际业务场景中真正有用。

- 为翻译员、分析师、研究员或助手等角色定义 AI 员工
- 在界面和工作流中实现 AI 与人类的无缝协作
- 确保 AI 使用安全、透明并可根据您的业务需求定制

### 3. 所见即所得，极其易用

在支持开发复杂业务系统的同时，NocoBase 保持了简单直观的用户体验。

- 一键切换使用模式和配置模式
- 页面作为画布来排列块和操作，类似于 Notion
- 配置模式专为普通用户设计，而不仅仅是程序员

### 4. 一切都是插件，专为扩展而设计

添加更多无代码功能永远无法覆盖所有业务案例。NocoBase 通过其基于插件的微内核架构构建，用于扩展。

- 所有功能都是插件，类似于 WordPress
- 插件安装后即可使用
- 页面、块、操作、API 和数据源都可以通过自定义插件扩展

## 安装方法

NocoBase 支持三种安装方法：

- **使用 Docker 安装（推荐）**：适用于无代码场景，无需编写代码。升级时，只需下载最新镜像并重启。
- **从 create-nocobase-app CLI 安装**：项目的业务代码完全独立，支持低代码开发。
- **从 Git 源代码安装**：如果您想体验最新未发布版本，或想参与贡献，需要对源代码进行更改和调试，建议选择此安装方法，需要较高的开发技能，如果代码已更新，可以 git pull 最新代码。
