# AI Agents for Beginners

这是一个由Microsoft提供的课程，旨在帮助初学者学习构建AI代理（AI Agents）。课程包含12个详细的课程，每个课程都涵盖AI代理构建的基础知识和高级主题。

## 功能

- **全面的课程内容**：从AI代理的基础概念到高级设计模式，包括工具使用、代理式RAG、信任构建、多代理系统等。
- **实践代码示例**：每个课程都提供Python代码示例，支持Azure AI Foundry和GitHub Models（免费）。
- **多语言支持**：课程支持多种语言，包括中文。
- **视频和额外资源**：每个课程都有配套视频和额外学习资源。
- **社区支持**：加入Azure AI Foundry Discord社区获取帮助。

## 用法

1. **开始学习**：访问[GitHub仓库](https://github.com/microsoft/ai-agents-for-beginners)并fork项目。
2. **环境设置**：参考[课程设置指南](/microsoft/ai-agents-for-beginners/blob/main/00-course-setup/README.md)配置Azure AI Foundry或GitHub Models。
3. **按顺序学习**：从第1课开始，逐步学习AI代理的概念和构建方法。
4. **运行代码**：使用提供的代码示例实践每个主题。
5. **加入社区**：在[Discord](https://aka.ms/ai-agents/discord)中与其他学习者交流。

课程适合对生成式AI有基础了解的开发者，如果你是新手，建议先学习[Generative AI For Beginners](https://aka.ms/genai-beginners)课程。
