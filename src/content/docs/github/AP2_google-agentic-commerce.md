# AP2 - Agent Payments Protocol

项目地址: https://github.com/google-agentic-commerce/AP2

## 项目简介

这个仓库包含了Agent Payments Protocol (AP2)的代码示例和演示。AP2旨在构建AI驱动支付的安全和互操作未来。

## 主要特性

- 使用Agent Development Kit (ADK)和Gemini 2.5 Flash。
- 协议不要求使用这些工具，但示例中使用了。
- 包含各种代理和服务器的演示。

## 项目文件结构

- samples/ 目录包含演示场景。
- samples/python/src 包含源代码。

## 安装与运行

1. 先决条件：Python 3.10+, uv。
2. 设置：配置API密钥。
3. 运行场景：bash samples/python/scenarios/your-scenario-name/run.sh

## 快速使用示例

导航到仓库根目录，运行脚本。
