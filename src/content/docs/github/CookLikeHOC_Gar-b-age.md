# 像老乡鸡那样做饭

像老乡鸡那样做饭。主要部分于2024年完工，非老乡鸡官方仓库。文字来自《老乡鸡菜品溯源报告》，并做归纳、编辑与整理。

## 主要特性

- 包含老乡鸡的所有菜品溯源报告。

- 分类整理：主食、凉拌、卤菜、早餐、汤、炒菜等。

- 支持Docker部署。

## 用法

- 查看docs目录中的菜品。

- 运行Docker：检查docker_support目录。

- 访问网页端：https://cooklikehoc.soilzhu.su

## 贡献

1. Fork 该仓库。
2. 创建特性分支：`git checkout -b feature/new-cook`.
3. 编写代码并通过测试。
4. 提交 PR 并阐明改动动机。

有任何问题，可以提交 Issue 或参与讨论。
