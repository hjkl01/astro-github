# asm-lessons_FFmpeg

## 功能

FFmpeg Assembly Language Lessons 是一个教育项目，旨在教授在FFmpeg中编写汇编语言的基础知识。通过一系列课程和练习，帮助学习者理解计算机底层操作，并为贡献FFmpeg代码做准备。

## 用法

1. 克隆仓库：`git clone https://github.com/FFmpeg/asm-lessons.git`

2. 阅读README.md以了解项目概述。

3. 浏览lesson_01、lesson_02等文件夹中的课程内容。

4. 加入Discord服务器讨论：https://discord.com/invite/Ks5MhUhqfB

**先决条件：**

- C语言知识，特别是指针。

- 高中数学基础。

通过这些课程，您将能够为FFmpeg项目贡献代码。
