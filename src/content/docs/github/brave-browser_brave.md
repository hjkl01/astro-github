# brave-browser

## 项目地址
[https://github.com/brave/brave-browser](https://github.com/brave/brave-browser)

## 概述
Brave Browser 是基于 Chromium 的隐私优先浏览器，旨在提供更快、更安全、更有收益的上网体验。

## 主要特性
- **隐私保护**：内置广告拦截、追踪器拦截、HTTP Strict Transport Security (HSTS) 预加载，默认开启 HTTPS Everywhere。
- **Brave Rewards**：通过观看隐私友好的广告赚取 Basic Attention Token (BAT)，支持将 BAT 捐赠给内容创作者或提现。
- **Tor 浏览**：支持 Tor 路由，提供匿名浏览功能。
- **内存与性能监控**：实时显示内存、CPU、网络使用情况，帮助用户优化资源。
- **多平台支持**：可在 Windows、macOS、Linux、Android、iOS 上运行。
- **Chrome 插件兼容**：支持安装大多数 Chrome 扩展。
- **同步功能**：通过 Brave Sync 同步书签、历史、密码等数据。

## 功能概述
1. **广告与追踪拦截**  
   - 默认开启；可在 `Settings > Shields` 自定义拦截级别。
2. **Brave Rewards**  
   - 在 `Rewards` 页面设置支付地址；开启后可观看广告赚取 BAT。
3. **Tor 浏览模式**  
   - 在 `Settings > Privacy and Security > Tor` 打开，所有请求通过 Tor 代理。
4. **HTTPS Everywhere**  
   - 自动将 HTTP 请求升级为 HTTPS，提升安全性。
5. **插件管理**  
   - 通过 `chrome://extensions/` 安装 Chrome 插件。
6. **同步**  
   - 在 `Settings > Sync` 登录 Brave 账户即可同步多设备数据。

## 使用方法
1. **下载与安装**  
   - 前往 GitHub 页面，选择对应操作系统的 Release 下载，按提示安装。
2. **首次启动**  
   - 启动后浏览器会弹出隐私设置向导，建议保持默认设置。
3. **开启/关闭特性**  
   - 访问 `Settings` → `Shields` 进行广告拦截、跟踪器拦截等设置。
4. **使用 Brave Rewards**  
   - 进入 `Rewards` 页面，绑定钱包地址，开启广告观看即可开始赚取 BAT。
5. **启用 Tor**  
   - 打开 `Settings` → `Privacy and Security` → `Tor`，切换为 “全局 Tor” 或 “仅 HTTPS”。
6. **安装插件**  
   - 打开 Chrome Web Store，搜索并安装所需插件；也可通过 `chrome://extensions/` 手动加载 CRX 文件。
7. **同步**  
   - 在 `Settings` → `Sync` 登录 Brave 账户，开启同步功能即可在不同设备间共享数据。

> **提示**：Brave 浏览器默认启用多项隐私保护功能，若想更改默认行为，请在 `Settings` 中自行调整。
