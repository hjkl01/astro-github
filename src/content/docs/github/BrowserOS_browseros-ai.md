# BrowserOS

BrowserOS是一个开源的chromium fork，能够在浏览器中本地运行AI代理。隐私优先，是ChatGPT Atlas、Perplexity Comet、Dia的替代品。

## 主要特性

- **AI代理本地运行**：AI代理在您的浏览器中运行，而不是云端。

- **隐私优先**：使用自己的API密钥或本地模型。数据留在您的计算机上。

- **开源**：完全开源，了解底层发生的一切。

- **熟悉的界面**：与Google Chrome相同的熟悉界面，支持所有扩展。

## 用法

1. 下载BrowserOS。

2. 导入Chrome数据（可选）。

3. 连接AI提供商（OpenAI, Anthropic, 或本地模型通过Ollama/LMStudio）。

4. 开始自动化！

## 快速开始

下载并安装BrowserOS，然后连接您的AI提供商。
