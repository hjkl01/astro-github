## 项目简介

KOReader.patches 是由 sebdelsol 维护的一个 KOReader 用户补丁集合。这些补丁全部兼容并协同工作，旨在增强 KOReader 电子书阅读器的功能和用户体验。KOReader 是一个开源的电子书阅读软件，支持多种格式的电子书。

## 主要功能

该项目包含多个用户补丁，每个补丁针对 KOReader 的特定功能进行改进：

- **UI 字体更改**：允许用户自定义界面字体。
- **状态栏增强**：包括薄状态栏章节标记、状态栏预设循环、状态栏颜色更改、紧凑模式改进等。
- **文件管理器改进**：显示标题栏信息、隐藏上访文件夹、隐藏空文件夹等。
- **屏幕保护功能**：添加章节信息、封面选项等。
- **其他实用补丁**：如菜单大小适配、参考页数修复、隐藏下划线等。

所有补丁都经过精心设计，确保彼此兼容，不会造成冲突。

## 使用方法

1. **安装补丁**：请参考 [KOReader 用户补丁安装指南](https://koreader.rocks/user_guide/#L2-userpatches) 来安装这些补丁。

2. **配置选项**：每个补丁在 KOReader 的设置菜单中都有相应的配置选项。例如：
   - UI 字体设置位于 **设置 > UI 字体**
   - 状态栏相关设置位于 **设置 > 状态栏**
   - 文件管理器设置位于 **导航 > 标题栏** 或 **导航 > 马赛克和详细列表设置**

3. **版本要求**：某些补丁需要特定的 KOReader 最低版本，请在使用前检查项目说明。

4. **更新补丁**：项目提供了更新补丁的功能，位于 **更多工具 > 更新 sebdelsol/KOReader.patches**。
