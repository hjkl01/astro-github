# BillionMail for aaPanel

**项目地址**：<https://github.com/aaPanel/BillionMail>

## 1. 项目简介  
BillionMail 是一款专为 aaPanel 设计的邮件服务器部署脚本，集成了 Postfix、Dovecot、Rspamd 等组件，支持 SMTP、IMAP/POP3、垃圾邮件过滤、反垃圾、病毒扫描等完整邮件功能，一键式安装与管理。

## 2. 主要特性  
- **一键安装**：在 aaPanel 中只需点击“安装”按钮，即可完成邮件服务器的完整部署。  
- **多协议支持**：SMTP、IMAP、POP3，支持 SSL/TLS 加密。  
- **反垃圾邮件**：集成 Rspamd，支持黑名单、白名单、低评分拦截。  
- **病毒扫描**：可配置 ClamAV 进行附件扫描。  
- **域名和别名管理**：通过 aaPanel 界面可轻松添加/删除域名、别名及虚拟邮件。  
- **使用限制**：支持邮箱容量、发送速率、邮箱数目等配额配置。  
- **邮件日志**：提供日志查看和错误排查。  

## 3. 功能概览  
| 功能 | 说明 |
|------|------|
| 账号管理 | 创建/删除邮箱账号，设置密码、配额、邮箱别名。 |
| 域名/别名 | 支持主域名、多域名、别名映射到目标邮箱。 |
| 反垃圾配置 | 黑名单、白名单、评分阈值、审计日志。 |
| 发送限制 | 设置每分钟/小时/天的发件数量。 |
| 日志分析 | 提供邮件流、错误、审计日志。 |
| SSL/TLS | 自动获取 & 续期 Let’s Encrypt 证书。 |
| 备份/迁移 | 导出邮件配置及内容，支持迁移。 |

## 4. 用法  

1. **在 aaPanel 后台安装**  
   - 登录 aaPanel → 访问插件市场 → 搜索 “BillionMail” → 点击 **安装**。  
   - 安装过程中会自动下载并配置 Postfix、Dovecot、Rspamd、ClamAV 等依赖。  

2. **配置域名**  
   - 在【邮件服务】中添加域名；若域名使用同一服务器的 DNS，系统会自动提示 MX 记录；否则手动在 DNS 解析处添加 MX 指向服务器 IP。  

3. **创建邮箱账号**  
   - 进入【邮箱账号】 → **添加账号**，填写用户名、密码、配额等；可批量导入 CSV。  

4. **管理反垃圾与病毒**  
   - 进入【安全策略】 → 配置 Rspamd 黑白名单、评分阈值；在【病毒扫描】设置 ClamAV。  

5. **查看邮件日志**  
   - 在【日志管理】中可以查看 SMTP、IMAP、Dovecot、Rspamd 的实时日志；支持搜索与导出。  

6. **客户端配置**  
   - 使用任何支持 IMAP/SMTP 的邮箱客户端（Outlook、Thunderbird、手机邮箱等），  
     - 收件服务器：`imap://<域名或IP>`（SSL/TLS）  
     - 发件服务器：`smtp://<域名或IP>`（SSL/TLS）  
     - 账号：`<邮箱>@<域名>`，密码即创建时设定的密码。  

## 5. 常见问题  
- **邮箱收发慢**：检查服务器网络、开启 ClamAV 后端的多线程扫描。  
- **反垃圾误判**：在 Rspamd 黑白名单中调整评分阈值或添加可信发件人。  
- **SSL 证书失效**：在【SSL/TLS】中手动重新获取 Let’s Encrypt 证书或更新自签证书。
