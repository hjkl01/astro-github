# CS 自学指南

## 项目简介

CS 自学指南是一个开源的计算机科学自学资源集合，由 PKUFlyingPig 创建。该项目旨在帮助自学者通过优质的开源课程和资源，系统地学习计算机科学知识，目标是在 2-3 年内成长为具备扎实基础的全能程序员。

## 主要功能

- **课程推荐**：涵盖算法、数据结构、操作系统、网络、数据库、人工智能等多个计算机科学领域，提供国内外名校的优质开源课程。
- **学习规划**：提供结构化的学习路径，帮助初学者从零开始逐步深入。
- **书籍推荐**：推荐经典的计算机科学教材和参考书籍。
- **项目实践**：鼓励通过实际项目巩固知识，涉及多种编程语言如 C/C++、Java、Python、Go 等。
- **社区交流**：支持页面评论和群聊建立，便于学习者交流讨论。

## 使用方法

1. **在线阅读**：访问官方网站 [csdiy.wiki](https://csdiy.wiki) 浏览指南内容。
2. **GitHub 查看**：在 [GitHub 仓库](https://github.com/PKUFlyingPig/cs-self-learning) 查看源码和文档。
3. **贡献内容**：如果您有补充或建议，可以提交 Pull Request 或在 Issues 中提出。
4. **加入交流**：在课程页面下方评论中加入学习群，或查看现有 Issues 中的群聊信息。

该指南适合计算机科学初学者、转行者或希望系统提升的开发者使用，通过自学优质资源，实现专业水平的知识积累。
