# OpenVoice

## 功能

OpenVoice 是一个由 MIT 和 MyShell 开发的即时语音克隆工具，具有以下主要功能：

- **准确的音色克隆**：能够准确克隆参考音色，并在多种语言和口音中生成语音。
- **灵活的语音风格控制**：允许对语音风格进行精细控制，如情感、口音，以及韵律、停顿和语调等参数。
- **零-shot跨语言语音克隆**：生成的语音语言或参考语音语言不需要在海量多语言训练数据集中出现。

OpenVoice V2 版本在此基础上提供了更好的音频质量、本地多语言支持（英语、西班牙语、法语、中文、日语和韩语），并允许免费商业使用。

## 用法

详细用法请参考项目的 [USAGE.md](https://github.com/myshell-ai/OpenVoice/blob/main/docs/USAGE.md) 文档。
