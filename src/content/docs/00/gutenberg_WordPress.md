# Gutenberg

## 项目简介

Gutenberg 是 WordPress 的块编辑器（Block Editor），它是 WordPress 5.0 及以上版本的默认内容编辑器。Gutenberg 允许用户使用“块”（Blocks）来构建和编辑内容，而不是传统的所见即所得（WYSIWYG）编辑器。

## 主要功能

- **块系统**：提供多种预定义块，如段落、标题、图像、列表、按钮、表格等，用户可以轻松添加和自定义这些块。
- **自定义块**：支持开发者创建自定义块，以扩展编辑器的功能。
- **响应式设计**：块可以根据设备屏幕大小自动调整布局。
- **实时预览**：编辑时实时预览内容的外观。
- **多媒体支持**：轻松嵌入图像、视频、音频和嵌入式内容。
- **协作编辑**：支持多用户协作编辑。
- **可扩展性**：通过插件和主题扩展功能。

## 用法

### 安装和启用

Gutenberg 是 WordPress 5.0+ 的核心组件，无需额外安装。如果使用较旧版本的 WordPress，可以从 WordPress 插件目录安装 Gutenberg 插件。

1. 登录 WordPress 后台。
2. 转到“插件” > “安装插件”。
3. 搜索“Gutenberg”。
4. 点击“安装”并“启用”。

### 基本使用

1. **创建新帖子或页面**：在 WordPress 后台，选择“帖子” > “添加新”或“页面” > “添加新”。
2. **添加块**：点击“+”按钮添加块，选择所需的块类型。
3. **编辑块**：点击块进行编辑，使用右侧的设置面板自定义块属性。
4. **保存和发布**：编辑完成后，点击“发布”按钮。

### 高级用法

- **自定义块**：使用 JavaScript 和 WordPress API 创建自定义块。
- **主题集成**：主题可以注册自定义块样式和支持。
- **插件扩展**：安装插件如 Advanced Custom Fields 或 Elementor 来增强功能。
