# Koodo Reader

**Project 地址**: https://github.com/koodo-reader/koodo-reader

---

## 主要特性

| 特性 | 描述 |
|------|------|
| 📚 **高品质阅读体验** | 采用自研排版引擎，支持双栏、单栏、全屏等多种阅读模式。|
| 🔌 **插件化架构** | 支持通过插件扩展功能，插件分为底层与业务插件，方便自定义。|
| 🛠️ **多平台同步** | Android、iOS、Web、桌面（macOS/Windows/Linux）统一数据同步。|
| 📦 **多格式支持** | 内置 EPUB、MOBI、PDF、TXT、FB2 等常见电子书格式支持。|
| 🔎 **全文搜索、书签** | 支持全文搜索、书签、笔记等功能，定位阅读进度。|
| 🎨 **自定义主题与字体** | 可切换多种主题、高级主题自定义，支持多字库与字体。|
| 💆 **多音频朗读** | 集成 TTS 引擎（Android、iOS）及支持本地离线音频。|
| 📱**离线离线** | 离线阅读、离线搜索、离线图片查看。|
| ⚙️ **高度可配置** | 手势配置、侧面菜单、音量键切换页码等弹性设置。|

## 核心功能

- **书架管理**：快速查看、分类、搜索书籍。  
- **云读**：无缝云端同步阅读进度、笔记、书签。  
- **高级阅读功能**：块标题与引用、阅读涂鸦、音频点读。  
- **插件系统**：通过插件实现主题切换、字典、脚本等多种扩展。  
- **脚本化**：开放脚本接口（JavaScript / Kotlin / Swift）可以自定义行为。  
- **多语言支持**：APP 内置多语言切换，校验书籍语言自动匹配对应字体。  

## 使用方法

1. **下载安装**  
   - Android: 通过 Google Play / APK 安装。  
   - iOS: 通过 App Store 安装。  
   - Web: 访问 https://koodo.me 或自行部署。

2. **导入电子书**  
   - 通过文件管理器导入，或通过共享/云端同步。  
   - 可直接在阅读器中选择**导入书籍**。

3. **阅读设置**  
   - 在左侧/右侧侧栏打开**阅读设置**。  
   - 选择主题、字体大小、排版方式、夜间模式等。

4. **插件与扩展**  
   - 打开**插件中心**，搜索并安装你需要的插件。  
   - 通过 **设置 → 插件** 对插件进行配置。

5. **云同步**  
   - 登录 GitHub / Google / 阿里云账号，开启同步功能。  
   - 同步会同步书签、阅读位置、笔记、漫画分段等。

6. **其他功能**  
   - **搜索**：使用侧面搜索框搜索全文。  
   - **书签**：长按章节标题或页面，设置书签。  
   - **音频朗读**：在阅读模式下点击右上角麦克风图标启动朗读。  
   - **短链接**：通过复制章节链接快速分享。  

> **提示**：若需自定义开发，请参考项目根目录下的 `docs/development.md`，其中包含插件开发、脚本化、贡献指南等内容。
