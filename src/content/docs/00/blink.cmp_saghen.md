# blink.cmp

## 项目简介

blink.cmp 是一个高性能、开箱即用的 Neovim 补全插件。它支持 LSP、命令行、签名帮助和代码片段，并使用可选的自定义模糊匹配器来提供拼写错误抵抗能力。该插件通过可插拔的源（LSP、缓冲区、代码片段等）、基于组件的渲染和动态配置提供可扩展性。

## 主要功能

- **开箱即用**：无需额外配置即可工作
- **高性能**：每次按键更新（异步 0.5-4ms，单核）
- **拼写错误抵抗模糊匹配**：使用自定义模糊匹配器，支持频率和邻近度奖励
- **广泛的 LSP 支持**：跟踪 LSP 功能支持
- **代码片段支持**：原生 `vim.snippet`（包括 `friendly-snippets`）、`LuaSnip` 和 `mini.snippets`
- **外部源支持**：社区源和 `nvim-cmp` 源的兼容层
- **自动括号支持**：基于语义标记
- **签名帮助**：实验性，选择性启用
- **命令行补全**
- **终端补全**：仅限 0.11+（尚无 shell 补全源，欢迎贡献！）

## 安装

请访问 [官方文档网站](https://cmp.saghen.dev/installation) 获取详细的安装说明和配置选项。

## 基本用法

安装后，blink.cmp 默认启用补全功能。你可以通过以下方式进行配置：

- 在 Neovim 配置中设置选项
- 使用动态配置进行自定义
- 添加额外的源和组件

更多配置细节请参考 [配置文档](https://cmp.saghen.dev/configuration/general)。
