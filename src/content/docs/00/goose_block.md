# goose

## 项目简介

goose 是一个开源、可扩展的 AI 代理工具，它超越了简单的代码建议，能够自动化复杂的开发任务。goose 可以从头构建整个项目、编写和执行代码、调试失败、编排工作流，并与外部 API 交互——所有这些都以自主方式进行。

## 主要功能

- **自动化开发任务**：从原型设计到完善现有代码，goose 能够处理复杂的工程管道。
- **多模型支持**：与任何大型语言模型 (LLM) 配合使用，支持多模型配置以优化性能和成本。
- **MCP 服务器集成**：无缝集成 MCP (Model Context Protocol) 服务器。
- **跨平台可用**：提供桌面应用和命令行界面 (CLI)，适用于不同开发环境。
- **自主执行**：能够独立安装、执行、编辑和测试代码，无需过多人工干预。

## 使用方法

### 安装

访问 [官方文档](https://block.github.io/goose/docs/getting-started/installation) 获取详细的安装指南。goose 支持多种平台，包括 Linux、macOS 和 Windows。

### 快速开始

1. **配置 LLM**：设置你选择的 LLM 提供商（如 OpenAI、Anthropic 等）。
2. **启动 goose**：使用 CLI 或桌面应用启动代理。
3. **描述任务**：向 goose 描述你需要完成的任务，例如“构建一个简单的 Web 应用”或“调试这个函数”。
4. **自主执行**：goose 将分析任务、编写代码、执行命令并处理错误。

### 示例用法

- **构建项目**：`goose build my-project` - goose 会从头创建项目结构、编写代码并运行测试。
- **代码编辑**：`goose edit file.js` - 让 goose 修改指定文件。
- **调试**：`goose debug error.log` - 分析错误日志并提供修复建议。
