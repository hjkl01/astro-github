# v2rayN（2dust/v2rayN）

- **GitHub 地址**: <https://github.com/2dust/v2rayN>

## 项目概述

v2rayN 是一款基于 Windows 的 V2Ray 客户端，提供图形化界面、订阅管理、自动更新、路由规则、DNS 解析等功能。它支持 V2Ray 核心以及多种代理协议（Shadowsocks、Trojan、Socks5 等），并能在系统层面切换代理。

## 主要特性

| 功能 | 说明 |
|------|------|
| **多协议支持** | Vmess、Shadowsocks、Trojan、Socks5、VMess+TLS、VMess+WebSocket 等 |
| **订阅管理** | 支持一次性或周期性订阅，自动解析配置 |
| **自动更新** | 自动检测并下载最新版 v2rayN 与 V2Ray 核心 |
| **高级路由** | GeoIP、Domain、IP 规则；可自定义 `routing.json` |
| **DNS 解析** | 支持 DNS-over-HTTPS、DNS-over-TLS，自动切换 |
| **系统代理** | 自动配置系统代理，支持全局、PAC、直连模式 |
| **自定义脚本** | 通过 `config.json` 设定启动参数、日志、进程管理 |
| **多账户管理** | 同时管理多条代理线路，快速切换 |
| **跨平台** | 通过 Windows Forms/WinUI，兼容 Windows 7+，可在 WSL 或 Docker 运行 |
| **日志与监控** | 内置日志查看、性能统计、连接状态监控 |
| **插件扩展** | 支持插件，便于集成自定义功能 |

## 快速上手

1. **下载安装**  
   - 下载最新的 `v2rayN.zip`（或直接使用 GitHub Release）  
   - 解压后双击 `v2rayN.exe` 启动。

2. **导入配置**  
   - ① 订阅：`订阅 → 订阅管理 → 添加订阅 URL → 启动订阅`  
   - ② 直接导入：`导入 → V2Ray 或 Shadowsocks 配置文件`  
   - ③ 手动添加：`服务器 → 新建服务器`，填写地址、端口、UUID 等。

3. **启用代理**  
   - 在主界面勾选想使用的线路，点击 `启动`。  
   - 选中 `系统代理`，v2rayN 会自动在系统代理列表中插入入口。

4. **系统代理设置**  
   - Windows 10/11：`设置 → 网络与 Internet → 代理`，勾选 `使用代理服务器` 并填写 `127.0.0.1:1080`（默认端口）。  
   - 可选择 `全局模式` 或 `自动模式`（PAC）。

5. **高级配置**  
   - **路由规则**：`设置 → 路由设置`，可导入或手动编辑 `routing.json`。  
   - **DNS**：`设置 → DNS 设置`，选择 `DNS-over-HTTPS/TLS` 或自定义解析器。  
   - **自定义参数**：`设置 → 进阶设置`，可在 `config.json` 中添加 `args` 等字段。

6. **日志与监控**  
   - `日志 → 查看日志`，可实时监控连接状态。  
   - `统计 → 连接统计`，查看流量与延迟。

## 常见操作

| 操作 | 步骤 |
|------|------|
| **切换线路** | 主界面点击对应线路 → `启动` |
| **重启 V2Ray** | `停止` → `启动` |
| **更新 v2rayN** | `帮助 → 检查更新` |
| **导出配置** | `导出 → 复制到剪贴板` 或 `导出为文件` |
| **清除缓存** | `设置 → 清除缓存` |

## 运行依赖

- Windows 7/8/10/11（32/64 位）  
- .NET 5/6/7（v2rayN 自带 .NET runtime，亦可单独安装）  
- V2Ray 核心（v2rayN 自动下载，或手动放置 `v2ray.exe`）。
