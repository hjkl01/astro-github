# ottomator-agents

## 简介

ottomator-agents 是 oTTomator Live Agent Studio 平台上托管的所有开源 AI 代理的集合。该平台由 oTTomator 开发，旨在为社区提供探索和学习 AI 代理的机会。

## 功能

- **开源代理库**：包含多种 AI 代理的源代码和工作流 JSON，支持各种用例，如 RAG（检索增强生成）、网络研究、聊天机器人等。
- **教育平台**：帮助用户学习如何构建和部署 AI 代理，同时提供实际价值。
- **社区驱动**：所有代理都是开源的，促进社区协作和学习。
- **多样化代理**：涵盖从基础 RAG 到高级多工具代理的各种实现，包括与 n8n、Pydantic AI 等工具的集成。

## 用法

1. **克隆仓库**：

   ```bash
   git clone https://github.com/coleam00/ottomator-agents.git
   ```

2. **探索代理**：查看各个子目录中的代理代码和工作流。每个代理都有独立的文件夹，包含实现细节。

3. **运行代理**：根据代理的 README 或代码注释，安装依赖并运行。许多代理使用 Python 或 n8n 工作流。

4. **构建自己的代理**：参考 `~sample-n8n-agent~` 或 `~sample-python-agent~` 文件夹作为起点。查看 [Developer Guide](https://studio.ottomator.ai/guide) 了解更多。

5. **使用平台**：访问 [Live Agent Studio](https://studio.ottomator.ai) 试用代理。新用户可获得免费 tokens。
