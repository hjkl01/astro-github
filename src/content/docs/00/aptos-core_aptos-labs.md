# Aptos Core

## 项目简介

Aptos Core 是 Aptos Labs 开发的 Layer 1 区块链项目，旨在通过更好的技术和用户体验为 Web3 带来范式转变。它使用 Move 编程语言，为开发者构建下一代应用程序提供了一个强大的平台。

## 主要功能

- **Layer 1 区块链**：提供高性能、可扩展的区块链基础设施。
- **Move 语言支持**：采用安全、可验证的 Move 编程语言，用于智能合约开发。
- **开发者友好**：专注于改善用户体验和开发者工具，支持快速构建 Web3 应用。
- **开源项目**：完全开源，鼓励社区贡献和协作。
