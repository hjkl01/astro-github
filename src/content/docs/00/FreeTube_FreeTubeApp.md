# FreeTube

## 简介

FreeTube 是一个开源的桌面 YouTube 播放器，注重隐私保护。它使用 Electron 构建，支持 Windows、macOS 和 Linux。FreeTube 允许用户在没有广告的情况下观看 YouTube 视频，并防止 Google 通过 cookies 和 JavaScript 跟踪用户。

## 功能

- **无广告观看视频**：观看 YouTube 视频时没有广告干扰。
- **隐私保护**：使用内置提取器或 Invidious API 获取数据，不使用官方 API，避免 Google 跟踪。
- **订阅管理**：无需账户即可订阅频道，本地存储订阅、播放列表和历史记录。
- **代理支持**：可连接外部代理，如 Tor。
- **主题和界面**：支持多种主题、迷你播放器、键盘快捷键等。
- **其他特性**：支持 SponsorBlock、DeArrow、视频截图、多个窗口等。

## 用法

1. **下载和安装**：
   - 从 [GitHub Releases](https://github.com/FreeTubeApp/FreeTube/releases) 或 [官方网站](https://freetubeapp.io/#download) 下载适合您操作系统的版本。
   - 支持 Windows 10+、macOS 12+ 和 Linux。

2. **启动应用**：
   - 安装后运行 FreeTube。

3. **搜索和观看视频**：
   - 在搜索栏中输入关键词或视频 URL。
   - 浏览趋势视频、订阅频道或播放列表。

4. **订阅频道**：
   - 在频道页面点击订阅按钮，订阅将本地存储。

5. **设置**：
   - 访问设置页面自定义主题、代理、提取器等。

注意：FreeTube 目前处于 Beta 阶段，可能存在一些 bug。建议使用 VPN 或 Tor 隐藏 IP。
