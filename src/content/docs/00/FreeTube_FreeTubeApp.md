# FreeTube – 适用于桌面的 YouTube 客户端

**GitHub 地址**: [https://github.com/FreeTubeApp/FreeTube](https://github.com/FreeTubeApp/FreeTube)

---

## 主要特性

- **完全离线**  
  运行在 Electron 桌面应用上，不依赖浏览器。

- **无跟踪与无广告**  
  全部请求通过本地播放器与自行解析的 YouTube API 进行，不收集用户数据，屏蔽所有广告。

- **自定义主题与黑暗模式**  
  支持全局主题切换（浅色/深色/自定义 CSS），并可在配置文件中自定义色彩和布局。

- **多语言支持**  
  包含多语言界面（如中文、英文、德语、西班牙语等），可以在设置中切换。

- **订阅与播放列表**  
  完整管理频道订阅和自定义播放列表；支持批量添加、删除、批量更新。

- **离线下载**  
  可将视频或音频以 MP4 / MP3 等格式下载到本地；支持批量下载、下载队列与下载管理。

- **搜素与推荐**  
  内置搜索引擎支持关键词、标签、频道过滤；自动显示热搜、推荐视频。

- **在本地保存观看记录**  
  视频播放记录、观看历史、收藏表单与观看位置等全部保存在本地数据库（SQLite）。

- **直播和 DVR**  
  支持观看直播及录制截图、直播聊天记录（可选导出）。

- **便携式与安装包**  
  提供 Windows（.exe、.msi）、macOS（.dmg）、Linux（AppImage、.deb、.rpm）等多平台安装包。

- **多音轨与字幕**  
  自动检测并切换可用字幕与多音轨；支持自定义字幕文件（.srt、.vtt）上传。

- **可扩展插件**  
  通过插件系统（如 IFrame API）可扩展功能（下拉列表、消息提醒等）。

- **安全审计**  
  项目在GitHub上公开源代码，社区可审计代码与依赖，及时修补安全漏洞。

## 功能细节

| 功能 | 说明 |
|------|------|
| **播放视频** | 从搜索结果、订阅列表、播放列表中点击即可开始播放。 |
| **调整画质** | 使用播放器控制面板切换 144p‑1080p、4K 等画质。 |
| **设置窗口** | 可修改缓存目录、主题、启用/禁用广告屏蔽、Google 登录、外观布局。 |
| **下载后处理** | 下载后可转换为 MP3，设置文件名模板，自动提取音频。 |
| **隐私模式** | 开启后不保存播放记录、搜索历史、禁用第三方插件。 |
| **分享** | 通过“分享”按钮复制链接、QR码、发布到社交平台。 |

## 安装与使用

1. **下载安装包**  
   - Windows: 下载 `FreeTube.Setup.exe`，双击安装。  
   - macOS: 下载 `.dmg`，拖拽至 `Applications`。  
   - Linux: 任选 AppImage/DEB/RPM，或使用 `snap install freetube`。

2. **首次启动**  
   - 应用会创建本地数据库与缓存目录（默认路径可在设置中修改）。  
   - 可在 *设置 → 账户* 区域登录 Google 以同步收藏与订阅，亦可保持离线即可使用。

3. **使用搜索**  
   - 在顶部搜索栏输入关键词 → 回车 → 结果列表。  
   - 可按发布时间、受欢迎程度、当前流行度等排序。  

4. **管理订阅**  
   - 打开左侧面板 → 订阅 → 添加 → 直接粘贴频道链接或在搜索中添加。  
   - 右键频道 → “移除订阅”，或点击“编辑订阅列表”。

5. **播放与下载**  
   - 双击视频预览 → 播放。  
   - 播放界面左下角有 **下载** 图标，点击可选择下载质量与格式。  

6. **离线观看**  
   在下载完成后，文件会保存在 `~/FreeTube/Downloads`（Windows为 `C:\Users\<User>\FreeTube\Downloads`）。  
   - 也可在“下载”面板直接打开或导入到播放器。

7. **自定义主题**  
   - 设置 → 主题 → 选择色盘或上传自定义 CSS。  
   - 通过 *主题预览* 调整配色方案。  

8. **快捷键**  
   - `Ctrl+P`：暂停/播放  
   - `Ctrl+Q`：退出  
   - `Ctrl+Shift+S`：截取屏幕截图  
   - `Ctrl+← / →`：后退/前进一段时间  
   - `← / →`：后退/前进 10 秒  
   - `↑ / ↓`：音量上下  
   - `M`：静音  
   - `F`：全屏/退出全屏  

9. **更新**  
   - 开发者会不定期发布新版本，使用自带更新提示即可自动下载并安装。  
   - 也可手动访问 GitHub Releases 页面下载最新版。

> **提示**：在 *设置 → 隐私* 中启用“隐私模式”，将不再记录观看历史与搜索关键词，适合注重隐私的用户。

---

> 以上为 FreeTube 的主要功能与使用方法，帮助用户在桌面环境中获得符合隐私与无广告的 YouTube 体验。
