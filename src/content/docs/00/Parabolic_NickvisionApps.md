# Parabolic

Parabolic 是一个强大的 yt-dlp 前端，用于下载网络视频和音频。它支持多种格式，并提供用户友好的界面。

## 功能

- **强大的 yt-dlp 前端**：支持 [yt-dlp 支持的站点](https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md)。
- **多种格式支持**：下载视频为 mp4、webm，音频为 mp3、opus、flac 和 wav。
- **并发下载**：同时运行多个下载任务。
- **元数据和字幕**：支持下载视频元数据和字幕。
- **跨平台**：提供 GNOME (Linux) 和 WinUI (Windows) 版本。

## 用法

### 安装

- **Flathub (Linux)**: 从 [Flathub](https://flathub.org/apps/details/org.nickvision.tubeconverter) 安装。
- **Snap Store**: 从 [Snapcraft](https://snapcraft.io/tube-converter) 安装。
- **手动下载**: 从 [GitHub Releases](https://github.com/NickvisionApps/Parabolic/releases) 下载适用于 Windows 或 Linux 的版本。

### 使用

1. 启动 Parabolic 应用。
2. 在主界面输入视频 URL。
3. 选择下载格式（视频或音频）。
4. 配置选项，如质量、字幕等。
5. 点击下载开始任务。
6. 监控下载进度，支持多个并发下载。

### 法律免责

请注意，YouTube 和其他网站的视频可能受 DMCA 保护。Parabolic 的作者不鼓励或负责使用此应用违反这些法律的行为。
