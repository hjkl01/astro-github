# HuYaTv

## 项目描述

HuYaTv（现名FUSE）是一个聚合直播TV客户端，支持多种直播平台的内容聚合和观看。项目仅供学习与交流使用，所有内容来源于网页端，不涉及任何破解行为。

## 主要功能

- **直播观看**：支持虎牙、抖音、快手、哔哩哔哩、斗鱼、YY、网易CC、Twitch等平台的直播流。
- **弹幕支持**：部分平台支持实时弹幕显示。
- **画质选择**：提供多种画质选项。
- **收藏功能**：允许用户收藏喜欢的直播间。
- **推送通知**：支持直播推送提醒。
- **搜索功能**：内置搜索功能，方便查找直播内容。
- **设备兼容**：支持手机、平板、电视、车机等多设备观看。
