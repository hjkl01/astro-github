# free-llm-api-resources

## 项目简介

这是一个免费LLM（Large Language Model）推理资源的列表，提供通过API访问的服务。这些资源包括完全免费的提供者和提供试用积分的提供者，帮助开发者在不产生高额成本的情况下测试和使用各种LLM模型。

## 主要功能

- **免费提供者列表**：收集了多个提供免费API访问的平台，如OpenRouter、Google AI Studio、NVIDIA NIM等，每个平台都有详细的模型列表和使用限制。
- **试用积分提供者**：列出了提供初始试用积分的平台，用户可以免费获得一定额度的积分来测试服务。
- **模型覆盖**：支持多种主流LLM模型，包括Llama、Mistral、Qwen、DeepSeek等系列。
- **使用限制说明**：每个提供者都明确标注了请求频率、每日/每月限制等信息，帮助用户合理规划使用。

## 使用方法

1. **选择提供者**：根据需求浏览免费提供者或试用积分提供者列表。
2. **注册账户**：访问相应平台的网站，完成注册（部分平台需要手机号验证）。
3. **获取API密钥**：在平台控制台中生成API密钥。
4. **集成使用**：使用API密钥调用相应模型的推理接口。
5. **注意限制**：遵守每个平台的速率限制和使用条款，避免滥用导致服务被封禁。

## 注意事项

- 请勿滥用这些免费服务，否则可能导致服务不可用。
- 明确排除任何非合法的服务（如逆向工程现有聊天机器人）。
- 部分平台的数据可能用于训练，请在使用前检查隐私政策。

- OpenRouter：提供多种模型，限制为20次/分钟，50次/天
- Google AI Studio：Gemini系列模型，免费但有严格限制
- Mistral：多种Mistral模型，支持实验计划
