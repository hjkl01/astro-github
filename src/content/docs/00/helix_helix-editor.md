# Helix

## 简介

Helix 是一个后现代模态文本编辑器，受到 Kakoune 和 Neovim 的启发，用 Rust 编写。编辑模型很大程度上基于 Kakoune。

## 功能

- Vim-like 模态编辑
- 多重选择
- 内置语言服务器支持
- 通过 tree-sitter 实现智能、增量语法高亮和代码编辑

## 安装

请参考 [安装文档](https://docs.helix-editor.com/install.html)。

## 用法

Helix 是一个终端-based 编辑器。所有快捷键和键映射可以在 [文档](https://docs.helix-editor.com/keymap.html) 中找到。

基本用法类似 Vim，使用模态编辑：normal mode, insert mode 等。
