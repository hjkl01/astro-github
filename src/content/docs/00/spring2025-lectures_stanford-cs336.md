# Stanford CS336 Spring 2025 讲座资料

**项目地址**  
[https://github.com/stanford-cs336/spring2025-lectures](https://github.com/stanford-cs336/spring2025-lectures)

## 主要特性

| 领域 | 说明 |
|------|------|
| 讲座幻灯片 | 每节课的 PPT（PDF/HTML）以及对应的讲义 |
| 视频链接 | 录制好的讲座视频（YouTube/Zoom 链接） |
| 代码示例 | 每节课配套的 Python/Java 代码片段，示例程序 |
| 练习与作业 | 章节练习题、实验作业（Notebook / 代码） |
| 参考资料 | 课程推荐阅读、相关论文与书籍链接 |
| 版本控制 | 通过 Git 追踪讲座内容的演进，支持多版本对比 |

## 功能概览

- **课程导航**：按周/节区分目录，方便快速定位所需讲座。
- **多格式展示**：PDF、HTML 版幻灯片均可在线预览，支持下载。
- **代码执行**：部分文件为 Jupyter Notebook，直接在 Colab 或本地 Jupyter 运行。
- **教学资源**：含讲义、实验手册、作业说明与参考答案（仅供学习参考）。

## 使用方法

1. **克隆仓库**  
   ```bash
   git clone https://github.com/stanford-cs336/spring2025-lectures.git
   cd spring2025-lectives
   ```

2. **浏览讲座内容**  
   - 进入 `lectures/` 目录，按周号浏览对应文件夹。  
   - 打开 `slides/` 或 `notes/` 子文件夹，查看 PDF/Markdown 幻灯片。  
   - 若需观看视频，可在对应文件夹的 `README.md` 中找到链接。

3. **运行代码示例**  
   - 对于 `.ipynb` 文件，直接在 Jupyter Notebook / Colab 打开并执行。  
   - 对于 `.py` 或 `.java` 示例，复制到本地 IDE 运行。  

4. **完成练习/作业**  
   - 在 `assignments/` 目录下找到每个作业的描述文件。  
   - 根据说明完成代码或报告，并在 `solutions/` 目录查看参考答案（仅供学习）。

5. **贡献与反馈**  
   - 若发现错误或想补充资料，可提交 Issue 或 Pull Request。  
   - 请遵循项目的 `CONTRIBUTING.md` 指南。

> **提示**：若想获取最新的讲座更新，建议使用 `git pull` 定期同步仓库。
