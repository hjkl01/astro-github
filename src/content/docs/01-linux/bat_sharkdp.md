## 配置示例（`~/.config/bat/config`）

```toml
# 颜色主题
theme = "Nord"

# 行号显示
line-number = true
line-range = ":"

# 自动分页
paging = "auto"

# 其他
style = "plain"  # 或 "grid"、"side-by-side"
```
