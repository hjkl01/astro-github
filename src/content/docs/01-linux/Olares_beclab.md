# Olares

Olares 是一个开源的个人云操作系统，旨在让您在本地拥有和管理您的数字资产，而不是依赖公共云服务。它提供强大的开源替代方案，如 Ollama 用于托管 LLM、SD WebUI 用于图像生成，以及 Mastodon 用于构建无审查社交空间。

## 功能

- **企业级安全**：使用 Tailscale、Headscale、Cloudflare Tunnel 和 FRP 简化网络配置。
- **安全的应用生态**：沙盒确保应用隔离和安全。
- **统一文件系统和数据库**：自动扩展、备份和高可用性。
- **单点登录**：一次登录访问 Olares 中的所有应用。
- **AI 能力**：全面的 GPU 管理、本地 AI 模型托管和私有知识库，同时保持数据隐私。
- **内置应用**：包括文件管理器、同步驱动、保险库、阅读器、应用市场、设置和仪表板。
- **无缝访问**：使用专用客户端从移动设备、桌面和浏览器任何地方访问您的设备。
- **开发工具**：全面的开发工具，用于轻松的应用开发和移植。

## 用法

### 系统兼容性

Olares 已测试并验证在以下 Linux 平台上：

- Ubuntu 24.04 LTS 或更高版本
- Debian 11 或更高版本

### 设置 Olares

要开始在您的设备上使用 Olares，请按照 [入门指南](https://docs.olares.com/manual/get-started/) 进行逐步操作。

### 关键用例

- **边缘 AI**：本地运行先进的开源 AI 模型，包括大语言模型、计算机视觉和语音识别。
- **个人数据仓库**：安全存储、同步和管理您的文件、照片和文档。
- **自托管工作空间**：为您的团队构建免费的协作工作空间。
- **私有媒体服务器**：托管您的个人媒体集合的流媒体服务。
- **智能家居中心**：为您的物联网设备和家庭自动化创建中央控制点。
- **去中心化社交媒体**：安装如 Mastodon、Ghost 和 WordPress 等应用。
- **学习平台**：动手探索自托管、容器编排和云技术。

更多信息请访问 [官方网站](https://olares.com) 和 [文档](https://docs.olares.com)。
