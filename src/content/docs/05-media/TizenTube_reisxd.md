# TizenTube

TizenTube 是一个 TizenBrew 模块，用于增强您在三星 Tizen TV 上观看流媒体网站的体验。它通过移除广告并添加 SponsorBlock 支持来改善观看体验。

## 功能

- **广告拦截器 (Ad Blocker)**: 自动移除视频中的广告。
- **SponsorBlock 支持**: 自动跳过视频中的赞助片段。
- **视频速度控制 (Video Speed Control)**: 允许调整视频播放速度。
- **DeArrow 支持**: 替换 YouTube 缩略图和标题以提供更好的描述。
- **可自定义主题 (Customizable Themes)**: 支持自定义颜色主题。

## 用法

1. 从 [TizenBrew](https://github.com/reisxd/TizenBrew) 安装 TizenBrew 并按照说明进行设置。
2. 在模块管理器中添加 NPM 模块 `@foxreis/tizentube`。可以通过遥控器按 [绿色] 按钮访问模块管理器。

安装后，模块将自动应用于支持的流媒体网站，提供无广告的观看体验。
