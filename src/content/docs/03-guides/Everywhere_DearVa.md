# Everywhere

## 项目简介

Everywhere 是一个上下文感知的桌面 AI 助手，能够智能响应并无缝集成多个大型语言模型 (LLM) 和 MCP 工具。与传统 AI 工具不同，Everywhere 能即时感知和理解屏幕上的任何内容，无需截图、复制或切换应用。只需按快捷键，即可在任何地方获得所需的 AI 帮助。

## 主要功能

### 🌟 体验示例

- **故障排除专家**：遇到错误消息时，直接在错误消息旁调用 Everywhere，询问“这是什么错误？如何解决？”即可获得解决方案。
- **网页快速摘要**：浏览长篇文章时，在网页上调用 Everywhere，命令“给我一个 3 点摘要”即可快速获取要点。
- **即时翻译**：遇到外语文本时，直接在文本上调用 Everywhere，命令“翻译成英语”即可获得翻译。
- **邮件草稿协助**：撰写商务邮件时，在草稿上调用 Everywhere，命令“让这听起来更专业”即可优化内容。

## 使用方法

### 安装

1. 访问 [GitHub Releases](https://github.com/DearVa/Everywhere/releases/latest) 下载最新版本。
2. 对于 Windows：
   - 下载 `Everywhere-Windows-x64-Setup-vx.x.x.exe` (安装包) 或 `Everywhere-Windows-x64-vx.x.x.zip` (免安装包)。
3. 运行安装程序或解压并运行可执行文件。

### 基本使用

1. 安装后，设置键盘快捷键 (默认通常是某个组合键)。
2. 在任何应用程序中，按快捷键调用 Everywhere。
3. 在弹出的界面中输入您的查询，Everywhere 会根据当前屏幕上下文提供智能响应。
4. 支持文本输入，也即将支持语音输入。

### 配置

- 在设置中配置您的 LLM API 密钥 (如 OpenAI, Anthropic 等)。
- 自定义快捷键和界面偏好。
- 启用或禁用特定工具集成。
