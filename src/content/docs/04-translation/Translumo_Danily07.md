# Translumo 项目

**GitHub 项目地址：** [https://github.com/ramjke/Translumo](https://github.com/ramjke/Translumo)

## 主要特性

Translumo 是一个开源的实时屏幕翻译器，专为游戏、视频硬编码字幕、静态文本等场景设计。它结合多个 OCR 引擎提供高精度的文本识别，并支持多种翻译服务。核心特性包括：

- **高精度文本识别**：同时使用多个 OCR 引擎，通过机器学习模型评分并选择最佳结果。
- **游戏导向**：专为 PC 游戏实时翻译设计，但适用于屏幕上的任何应用。
- **低延迟**：优化以减少系统影响并最小化文本出现到翻译之间的延迟。
- **集成现代 OCR 引擎**：Windows OCR（推荐）、Tesseract 5.2（遗留）、EasyOCR（遗留）。
- **可用翻译器**：DeepL（推荐）、Google Translate、Yandex Translate、Naver Papago。
- **支持识别语言**：英语、俄语、日语、中文（简体）、韩语。
- **支持翻译语言**：英语、俄语、日语、中文（简体）、韩语、法语、西班牙语、德语、葡萄牙语、意大利语、越南语、泰语、土耳其语、阿拉伯语、希腊语、巴西葡萄牙语、波兰语、白俄罗斯语、波斯语、印尼语、保加利亚语、捷克语、丹麦语、爱沙尼亚语、芬兰语、匈牙利语、立陶宛语、拉脱维亚语、荷兰语、罗马尼亚语、斯洛伐克语、斯洛文尼亚语、瑞典语、乌克兰语。
- **开源免费**：基于 Apache-2.0 许可。

## 系统要求

### 最低要求（使用 Tesseract 和 Windows OCR）

- Windows 10 版本 2004（构建 19041）或更高版本，或 Windows 11
- DirectX 11 兼容 GPU
- 2 GB RAM

### 最低要求（使用 EasyOCR）

- 支持 CUDA SDK 11.8 的 NVIDIA GPU（GTX 750、8xxM、9xx 系列或更新）
- 8 GB RAM
- 至少 5 GB 可用存储空间

## 用法

1. **下载与安装**：
   - 访问 GitHub 项目页面，下载最新版本的 ZIP 文件（例如 Translumo_1.0.2.zip）。
   - 解压存档并运行 Translumo.exe。

2. **启动与配置**：
   - 打开 Translumo 应用。
   - 按 Alt+G 打开设置。
   - 选择语言：源语言用于 OCR，翻译语言用于目标。
   - 选择文本识别引擎（推荐使用 WindowsOCR）。
   - 定义捕获区域：按 Alt+Q 并在屏幕上选择一个区域。
   - 运行翻译：按 ~ 键。

### 推荐 OCR 引擎

- 推荐仅使用 WindowsOCR。Tesseract 旧且慢，产生许多错误。EasyOCR 更慢，需要大量资源，且经常出现错误。

### 选择最小捕获区域

减少捕获区域以降低从背景拾取随机字母的机会。较大的帧需要更长时间处理。

### 使用代理列表避免翻译服务阻塞

一些翻译器可能会阻止发送许多请求的客户端。在“语言 → 代理”选项卡下配置个人或共享 IPv4 代理（1-2 个通常足够）。应用将轮换代理以减少来自单个 IP 的请求。

### 在游戏中使用无边框或窗口模式（非全屏）

这些模式对于正确显示翻译覆盖是必需的。如果您的游戏不支持它们，请使用 Borderless Gaming 等工具。

## 故障排除

- **获取“捕获屏幕失败”或按翻译开始后无反应**：确保目标窗口处于活动状态。如有必要，重启 Translumo 或重新打开目标窗口。
- **无边框/窗口模式已设置，但翻译窗口在游戏下方**：游戏运行并聚焦时，按默认 Alt+T 热键隐藏和显示翻译窗口。
- **EasyOCR 包下载失败**：尝试在连接 VPN 时重新安装。
- **热键不起作用**：其他应用程序可能正在拦截热键。
- **文本检测失败（TesseractOCREngine）**：确保应用程序路径仅包含拉丁字母。
