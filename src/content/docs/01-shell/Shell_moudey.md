# Shell (moudey/Shell)

> 项目地址: <https://github.com/moudey/Shell>

## 概述

Shell 是一个强大的 Windows 文件资源管理器上下文菜单管理器。它允许用户自定义和扩展 Windows 资源管理器的右键菜单，添加新的菜单项、修改或删除现有项，并提供高度可定制的界面。

## 主要特性

- **轻量级且便携**：占用资源少，易于使用和部署。
- **完全可定制外观**：支持自定义颜色、图标和布局。
- **添加自定义项**：支持添加子菜单、菜单项和分隔符。
- **修改或删除现有项**：可以修改或移除系统或第三方软件添加的上下文菜单项。
- **全面的文件系统支持**：支持所有文件系统对象，包括文件、文件夹、桌面和任务栏。
- **表达式语法支持**：内置函数和预定义变量，支持复杂逻辑。
- **丰富的视觉元素**：支持颜色、字形、SVG、嵌入式图标和图像文件（如 .ico、.png 或 .bmp）。
- **搜索和过滤**：内置搜索和过滤功能。
- **复杂嵌套菜单**：支持多列显示和复杂的嵌套菜单结构。
- **纯文本配置**：使用纯文本文件进行配置，易于编辑。
- **极少资源占用**：高效运行，无性能负担。
- **无限制**：功能完全开放，无任何限制。

## 功能概览

| 功能               | 说明                                   |
| ------------------ | -------------------------------------- |
| **上下文菜单管理** | 自定义 Windows 资源管理器右键菜单      |
| **自定义命令**     | 添加访问网页、文件、文件夹的自定义命令 |
| **界面定制**       | 修改或移除系统/第三方添加的菜单项      |
| **文件系统集成**   | 支持所有类型的文件和文件夹操作         |
| **高级配置**       | 通过表达式和脚本实现复杂逻辑           |

## 要求

- Microsoft Windows 7/8/10/11

## 安装和用法

1. **下载**：从 [官方网站](https://nilesoft.org/download) 下载最新版本。
2. **安装**：运行安装程序或解压便携版。
3. **配置**：使用纯文本配置文件自定义菜单项。
4. **运行**：重启资源管理器或系统以应用更改。
