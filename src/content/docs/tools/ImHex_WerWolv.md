## 功能介绍

ImHex 是一个功能强大的十六进制编辑器，专为逆向工程师、程序员和在深夜工作时重视视网膜健康的用户设计。它提供了丰富的功能来分析、编辑和可视化二进制数据。

### 主要功能

- **功能丰富的十六进制视图**：
  - 字节修补和补丁管理
  - 无限撤销/重做
  - 支持多种格式复制（字节、十六进制字符串、各种编程语言数组等）
  - 字符串和十六进制搜索
  - 跳转功能（从开始、结束或当前光标位置）
  - 彩色高亮显示（可配置的前景规则、背景高亮使用模式、查找结果和书签）

- **自定义模式语言**：
  - C++-like 语法用于解析和突出显示文件内容
  - 支持数组、指针、结构体、联合体、枚举、位域等
  - 自动基于 MIME 类型和魔数加载
  - 可视化多种数据类型（图像、音频、3D 模型等）

- **主题和界面**：
  - 默认暗模式，保护视力
  - 可自定义颜色和样式
  - 支持自定义字体
  - 多工作区和自定义布局
  - 可分离窗口

- **数据处理**：
  - 数据检查器：支持多种数据类型解释（整数、浮点、字符串、时间戳等）
  - 基于节点的预处理器：修改、解密和解码数据
  - 导入/导出支持（Base64、IPS 补丁、Markdown 报告等）

- **数据源支持**：
  - 本地文件（支持大文件高效加载）
  - 原始磁盘和分区
  - GDB 服务器（访问运行进程的 RAM）
  - Intel Hex 和 Motorola SREC 数据
  - UDP 数据包
  - 进程内存
  - 通过 SSH/SFTP 的远程文件

- **搜索和分析**：
  - 多种搜索类型（字符串、序列、正则表达式、二进制模式、数值）
  - 数据哈希支持（CRC、MD5、SHA 等多种算法）
  - 差异比较和突出显示
  - 集成反汇编器（支持多种架构）
  - YARA 规则扫描

- **工具集成**：
  - 书签管理
  - 数据分析器（熵图、分布图等）
  - 各种实用工具（计算器、颜色选择器、文件工具等）
  - 内置内容更新器（下载模式文件、库等）

## 用法

1. **安装**：
   - 从 [GitHub Releases](https://github.com/WerWolv/ImHex/releases) 下载最新版本
   - 支持 Windows、macOS、Linux 等平台
   - 详细安装指南见 [INSTALL.md](https://github.com/WerWolv/ImHex/blob/master/INSTALL.md)

2. **基本使用**：
   - 打开文件：文件 -> 打开，选择要分析的文件
   - 十六进制视图：查看和编辑字节数据
   - 应用模式：使用内置或自定义模式解析数据结构
   - 搜索：使用搜索功能查找特定数据
   - 导出：保存修改或导出为不同格式

3. **高级功能**：
   - 使用数据处理器节点处理数据
   - 应用 YARA 规则扫描漏洞
   - 使用反汇编器分析代码
   - 创建书签标记重要区域

4. **自定义**：
   - 下载额外模式文件从 [ImHex-Patterns](https://github.com/WerWolv/ImHex-Patterns)
   - 创建自定义主题和布局
   - 开发插件扩展功能

ImHex 适合逆向工程、软件分析、安全研究等场景，提供直观且强大的二进制数据处理能力。
