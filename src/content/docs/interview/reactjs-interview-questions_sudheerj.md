# ReactJS 面试题集（sudheerj）

项目地址: [https://github.com/sudheerj/reactjs-interview-questions](https://github.com/sudheerj/reactjs-interview-questions)

## 主要特性

- **结构化目录**
  - 按主题划分（基础、组件、生命周期、Hooks、性能等）。
  - 每个主题下都有对应的 `Questions`、`Answers`、`Examples` 三部分，便于快速定位。

- **问题与答案完整**
  - 提供从常见面试题到深层次技术考点的完整问答。
  - 通过代码示例展示关键概念，实现对答案的可验证性。

- **代码示例**
  - 每个答案后面都附带可直接运行的 React 示例。
  - 示例涵盖不同版本和最佳实践。

- **易于贡献**
  - 使用 Markdown 保存问题/答案，编辑、提交 PR 方便。
  - README 指明如何添加新问题、修改答案、格式规范。

## 用法

1. **浏览代码仓库**
   - `src/` 目录下按主题文件夹组织，如 `01-basics`、`02-components` 等。
   - 每个文件夹包含问题和答案的 markdown 文件。

2. **本地查看**
   - 克隆仓库后，在支持 Markdown 的编辑器中打开相应文件即可。
   - 或者使用 `npx live-server` 等工具直接预览。

3. **贡献新问题/答案**
   - 在对应主题文件夹下新建问题和答案文件。
   - 按照已有格式编写，提交 PR。

4. **搜索与筛选**
   - 通过 IDE 搜索 `Question:`, `Answer:` 等关键词快速定位。
   - 若需聚焦难度，可使用文件名或文件内注释标记。

> 该项目适合作为面试准备、技术复盘或团队内部知识库。祝你学习愉快！
