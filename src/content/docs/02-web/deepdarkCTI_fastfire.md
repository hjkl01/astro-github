# deepdarkCTI

## 项目简介

deepdarkCTI 是由 fastfire 创建的一个开源项目，旨在收集和整理来自深网（Deep Web）和暗网（Dark Web）的网络威胁情报（Cyber Threat Intelligence, CTI）来源。该项目帮助安全研究人员和组织监控网络威胁、分析威胁行为者的战术、技术和程序（TTPs），并提供战略、战术和操作层面的情报。

## 主要功能

- **情报来源收集**：项目整理了多种深网和暗网来源，包括：
  - Telegram 频道、群组和聊天
  - Discord 频道
  - 勒索软件团伙网站
  - 网络犯罪论坛和数据泄露相关论坛
  - 暗网市场
  - 漏洞利用数据库
  - Twitter 账户
  - RaaS（Ransomware as a Service）站点

- **分类整理**：情报按类别组织在不同的 Markdown 文件中，如商业服务、假冒商品、CVE 漏洞、网站篡改、Discord、漏洞利用、论坛、MaaS（Malware as a Service）、恶意软件样本、市场、方法、其他、钓鱼、勒索软件团伙、RAT（Remote Access Trojan）、搜索引擎、Telegram 信息窃取者、Telegram 威胁行为者、Twitter、Twitter 威胁行为者等。

- **社区支持**：项目维护者提供 Telegram 群组，用于讨论新来源提议和 CTI 研究技术。

## 用法

1. **浏览来源**：访问项目的 GitHub 仓库或官方网站 [https://www.deepdarkcti.com](https://www.deepdarkcti.com)，查看各个分类文件中的情报来源链接和描述。

2. **分析情报**：使用这些来源进行 OSINT（Open Source Intelligence）研究，识别威胁指标（IoCs）、威胁行为者 TTPs 和新兴威胁。

3. **贡献和讨论**：
   - 通过 Twitter [@fastfire](https://twitter.com/fastfire)、Telegram [@fastfire83](https://t.me/fastfire83) 或 Bluesky [fastfire.bsky.social](https://bsky.app/profile/fastfire.bsky.social) 请求加入 Telegram 群组。
   - 提交 Pull Request 或 Issue 来添加新来源。

4. **捐赠支持**：项目接受捐赠，用于维护和扩展资源。
