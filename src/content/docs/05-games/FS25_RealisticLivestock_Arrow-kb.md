# FS25_RealisticLivestock

## 项目简介

FS25_RealisticLivestock 是为 Farming Simulator 25 开发的模组，由 Arrow-kb 创建。该模组旨在为游戏添加更真实的牲畜管理机制，使牲畜的行为和生命周期更加贴近现实。

## 主要功能

- **年龄系统**：牲畜现在有年龄，从出生到成年再到老年，影响生产力和行为。
- **繁殖系统**：牲畜可以自然繁殖，包括怀孕期和出生过程。
- **健康和疾病**：牲畜可能生病，需要照顾以保持健康。
- **行为改进**：更真实的动物行为，如觅食、社交互动等。
- **经济影响**：年龄和健康状态影响牲畜的价值和产量。

## 安装和用法

1. 下载模组文件：从 GitHub 仓库下载最新的发布版本。
2. 安装到游戏：将模组文件夹复制到 Farming Simulator 25 的 mods 目录中（通常位于 `Documents/My Games/FarmingSimulator25/mods`）。
3. 启用模组：在游戏启动器中启用该模组。
4. 开始游戏：在新游戏或现有存档中，购买牲畜并体验更真实的养殖体验。

## 注意事项

- 确保游戏版本兼容（专为 FS25 设计）。
- 备份存档以防模组冲突。
- 查看仓库的 Issues 页面获取最新更新和已知问题。
