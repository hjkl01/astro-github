# Rime万象拼音输入方案

Rime万象拼音输入方案基于深度优化的词库和语法模型，支持标准版与增强版可选。词库通过AI筛选和语料辅助筛选，精干高效，配合全新语法模型，输入不再纠结。PRO版本支持10种双拼、6种辅助码，可扩展。支持混合编码输入，内置超级注释、带调全拼输入码显示、快符、候选手动排序、tips、首选成对符号包裹等功能扩展，大大增强使用体验。

## 功能特性

- **词库优化**：带声调的词库，经过AI和大基数语料筛选、加频，结合语言模型获得更准确的整句输出。
- **辅助码支持**（PRO版本）：支持直接辅助、间接辅助、声调辅助等多种辅助码输入方式，帮助快速定位生僻字或特定字词。
- **反查功能**：支持两分、多分、笔画三种打法，用于生僻字输入和反查。
- **混合编码**：支持字母、汉字、数字、特殊符号组合输入，以及英文混输。
- **扩展功能**：日期时间模板、Unicode输入、金额大写、计算器、快符、超级tips、首选格式化、成对符号包裹、英文候选格式化、辅助码提示、用户按需造词、无感造词、手动排序等。
- **自定义能力**：支持自定义短语、词库、按键设置等。

## 使用方法

1. **安装**：将方案文件置于Rime用户目录中，直接部署。
2. **切换输入类型**：输入斜杠引导指令切换双拼/全拼，如 `/flypy` 切换到小鹤双拼，`/pinyin` 切换到全拼。
3. **自定义配置**：使用 `wanxiang.custom.yaml` 进行补丁配置，支持patch schema、default、weasel等。
4. **更新脚本**：使用提供的更新脚本自动更新词库和配置。
5. **辅助码使用**：
   - 直接辅助：在双拼后追加辅助码，如 `vfj` 输入“镇”。
   - 间接辅助：在双拼后加 `/` 引导辅助码，如 `ni/re`。
   - 声调辅助：使用7890代表1234声，如 `ni9`。
6. **其他功能**：
   - 日期时间：`/sj` 输入时间，`/rq` 输入日期。
   - 符号：`/sx` 输入数学符号。
   - 计算器：`V3+5` 计算结果。
   - Tips：Ctrl+t 开关，`.` 上屏提示内容。

更多详情请参考项目README和自定义文件夹中的教程。
