# yay

## 功能

yay (Yet Another Yogurt) 是一个用于 Arch Linux 的 AUR (Arch User Repository) 助手。它允许用户轻松地从 AUR 安装、更新和管理软件包。

主要功能包括：

- 从 AUR 搜索和安装软件包
- 自动处理依赖关系
- 支持 pacman 的所有操作
- 彩色输出和交互式界面

## 用法

### 安装软件包

```bash
yay -S package_name
```

### 更新系统和 AUR 包

```bash
yay -Syu
```

### 搜索软件包

```bash
yay -Ss search_term
```

### 移除软件包

```bash
yay -R package_name
```

更多用法请参考官方文档：https://github.com/Jguer/yay
