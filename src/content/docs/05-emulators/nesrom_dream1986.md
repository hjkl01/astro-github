# nesrom

## 项目地址

[https://github.com/dream1986/nesrom](https://github.com/dream1986/nesrom)

## 项目概述

这是一个 NES 游戏合集，包含各种经典 NES 游戏 ROM 文件。由用户 dream1986 整理分享。

## 主要特性

- 收集了大量 NES 经典游戏
- 包括多种游戏类型，如动作、冒险、射击等
- 文件格式为 .nes，可用于 NES 模拟器

## 用法

1. 下载项目文件
2. 使用 NES 模拟器（如 FCEUX、VirtuaNES 等）加载 .nes 文件
3. 享受经典游戏

## 注意事项

- 请确保拥有原版游戏版权，仅用于个人学习和娱乐
- 使用合法的模拟器运行游戏
