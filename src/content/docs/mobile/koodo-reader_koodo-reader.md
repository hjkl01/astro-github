# Koodo Reader 项目

**GitHub 项目地址:** [https://github.com/koodo-reader/koodo-reader](https://github.com/koodo-reader/koodo-reader)

## 主要特性

Koodo Reader 是一款跨平台的电子书阅读器，支持多种格式和平台。主要特性包括：

- **格式支持**：EPUB (.epub)、PDF (.pdf)、DRM-free Mobipocket (.mobi) 和 Kindle (.azw3, .azw)、纯文本 (.txt)、FictionBook (.fb2)、漫画书档案 (.cbr, .cbz, .cbt, .cb7)、富文本 (.md, .docx)、超文本 (.html, .xml, .xhtml, .mhtml, .htm)。
- **平台支持**：Windows、macOS、Linux、Android、iOS 和 Web。
- **云同步**：支持 OneDrive、Google Drive、Dropbox、MEGA、pCloud、Yandex Disk、Box、FTP、SFTP、WebDAV、SMB 或对象存储同步和备份数据。
- **AI 功能**：AI 字典、摘要和翻译。
- **布局选项**：单列、双列或连续滚动布局。
- **其他功能**：文本到语音、翻译、字典、触摸屏支持、批量导入、书签、笔记、高亮、字体大小、字体家族、行间距、段落间距、背景颜色、文本颜色、边距和亮度调整、夜间模式和主题颜色、文本高亮、下划线、粗体、斜体和阴影。

## 主要功能

- **书籍管理**：轻松导入书籍，支持批量导入。
- **阅读体验**：自定义字体、颜色、布局等，实现个性化阅读。
- **注释功能**：添加书签、笔记和高亮。
- **同步备份**：跨设备同步阅读进度和数据。
- **多语言支持**：支持多种语言，包括中文、英文等。

## 用法

1. **安装**：
   - 桌面版：[下载](https://koodoreader.com/en/download)
   - Web 版：[访问](https://web.koodoreader.com)
   - Android 版：[下载](https://koodoreader.com/en/download)
   - iOS 版：[下载](https://koodoreader.com/en/download)
   - 使用 Scoop：`scoop bucket add extras; scoop install extras/koodo-reader`
   - 使用 Homebrew：`brew install --cask koodo-reader`
   - 使用 Docker：[安装指南](https://koodoreader.com/en/deploy-docker)

2. **开发**：
   - 克隆仓库：`git clone https://github.com/koodo-reader/koodo-reader.git`
   - 桌面模式：`yarn; yarn dev`
   - Web 模式：`yarn; yarn start`

3. **翻译**：
   - 编辑当前语言或添加新语言，参考项目 README 中的翻译指南。

该项目适合电子书爱好者，提供强大的阅读和个性化功能。
