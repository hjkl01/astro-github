# rime_wanxiang

## 项目简介

rime_wanxiang 是基于 RIME 输入法引擎的一个开源输入法配置项目，由用户 amzxyz 维护。该项目提供万象输入方案，支持多种中文输入方法，包括拼音、五笔等。

## 主要功能

- **多种输入方案**：支持拼音输入、五笔输入等多种编码方式。
- **自定义配置**：用户可以根据需要调整词库、快捷键等设置。
- **跨平台支持**：兼容 Windows、macOS、Linux 等操作系统。
- **开源词库**：基于开源词库，提供丰富的词汇覆盖。

## 使用方法

1. **安装 RIME 输入法**：
   - 下载并安装 RIME 输入法引擎（如小狼毫、鼠须管等）。

2. **下载配置**：
   - 从 GitHub 仓库 https://github.com/amzxyz/rime_wanxiang 下载配置文件。

3. **部署配置**：
   - 将下载的配置文件复制到 RIME 的用户配置文件夹（通常位于 `~/Library/Rime` 或 `%APPDATA%\Rime`）。
   - 重启 RIME 输入法以应用新配置。

4. **自定义设置**：
   - 编辑 `default.custom.yaml` 或其他配置文件以调整输入偏好。
   - 同步词库以获取最新更新。

## 注意事项

- 请确保 RIME 版本兼容。
- 备份原有配置以防数据丢失。
- 更多详细信息请参考项目 README 或 RIME 官方文档。
